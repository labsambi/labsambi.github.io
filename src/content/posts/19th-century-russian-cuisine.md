# 19世纪俄罗斯常见菜谱汇总

> 本文以**19世纪（1801–1900年）沙皇俄国**为核心，综合梳理了当时各阶层——从贵族宴席到农民餐桌——的典型菜肴、调味品、饮品及饮食习俗。参照的原始文献包括：Elena Molokhovets《给年轻主妇的礼物》(Подарок молодым хозяйкам，1861年初版，至1917年再版29次)、18–19世纪修道院食谱、以及同时代欧洲旅行者的记录。每道菜均标注社会阶级、做法概要、主要原料与历史背景。

## 一、汤类（Супы）

汤是俄罗斯饮食的灵魂。俄国谚语云："Щи да каша — пища наша"（白菜汤和粥是我们的食粮）。19世纪的俄罗斯汤品分为几大类型，既有古老的农家汤，也有受法国烹饪影响的精致清汤。由于东正教全年约有180–200天为斋戒日（禁肉、禁奶、禁蛋），大部分汤品都有斋戒版与荤食版两种做法。

### 1. Щи（Shchi）—— 白菜汤

| 项目 | 内容 |
|------|------|
| **类型** | 国民主食汤 |
| **社会阶级** | 全民（从沙皇到农民） |
| **起源** | 基辅罗斯时期（9世纪），逾千年历史 |
| **主要原料** | 卷心菜（新鲜或酸渍）、肉类（牛肉/猪肉）或蘑菇、洋葱、胡萝卜、香料根（防风根/欧芹根）、酸奶油（smetana）上桌时点缀 |

**做法概要：**
- **荤版（Богатые щи / 富汤）**：牛肉带骨熬制浓汤，加入切碎的鲜卷心菜或酸菜、炒过的洋葱和胡萝卜、土豆（19世纪中期后普及）。慢炖2–3小时后，上桌前加一勺酸奶油和新鲜莳萝（укроп）。
- **斋版（Постные щи / 斋汤）**：以蘑菇（干牛肝菌为佳）煮出素高汤替肉汤，不加酸奶油。有时加入腌黄瓜的卤水提酸。
- **灰白菜汤（Серые щи）**：用卷心菜最外层的粗糙绿叶制作，是贫农冬季主食。发酵后的外层菜叶有独特的酸味和韧性。

**历史背景：** Shchi是俄罗斯最古老的汤品，出现在书面记录中超过千年。俄国谚语见证了它的地位。19世纪贵族阶层中，shchi常在正式晚宴上以小陶瓷罐（горшочек）单独上桌；而农民版本的酸菜汤则可能一次煮一大锅，连续吃数日，反复加热后风味反而更浓——这在俄语中称为"суточные щи"（隔夜汤）。

### 2. Борщ（Borscht）—— 红菜汤

| 项目 | 内容 |
|------|------|
| **类型** | 甜菜根浓汤 |
| **社会阶级** | 全民（乌克兰及南俄地区的日常） |
| **起源** | 乌克兰/南俄，最早文字记录14–15世纪 |
| **主要原料** | 甜菜根、卷心菜、土豆、牛肉/猪肉、番茄（19世纪后期加入）、酸奶油、莳萝、大蒜、猪油（сало） |

**做法概要：**
- 将牛骨或猪骨与肉同炖出底汤。
- 甜菜根切丝后先煸炒（或加醋/柠檬汁）以保持鲜红色。
- 依次加入胡萝卜、洋葱、土豆、卷心菜慢炖。
- 出锅前加捣碎的生蒜和猪油（老派波兰-乌克兰做法，称为"затирка"），上桌配酸奶油、黑麦面包。

**历史背景：** 在19世纪，"борщ"一词主要指乌克兰版本的酸味甜菜浓汤，与纯俄式的shchi在食材上有所区分。随着帝国扩张和铁路建设，这道菜从乌克兰和南俄地区传播到圣彼得堡和莫斯科。Molokhovets的书（1861）中收录了多个版本，包括用格瓦斯（kvass）作汤底的斋戒版。

### 3. Уха（Ukha）—— 清炖鱼汤

| 项目 | 内容 |
|------|------|
| **类型** | 清汤鱼羹 |
| **社会阶级** | 全民（渔民与贵族的共同热爱） |
| **起源** | 古罗斯时期，原指任何肉汤，17世纪后专指鱼汤 |
| **主要原料** | 多种河鱼（鲈鱼окунь、梭鲈судак、江鳕налим、鲟鱼осётр）、洋葱、胡萝卜、月桂叶、黑胡椒粒、莳萝、伏特加（出锅前加一shot） |

**做法概要：**
- 经典做法要求至少用三种鱼：小鱼（如鮈鱼）熬汤后弃之取清汤；再加入大块的中型鱼（鲈鱼、梭鲈）；豪华版最后加鲟鱼肉块。
- 蔬菜仅用洋葱和胡萝卜，切大块，不炒（讲究清亮）。
- 慢火熬制，不用盖锅，忌用铁锅（多用搪瓷或铜锅）。
- 出锅时滴入少量伏特加以去腥提鲜，撒莳萝。

**历史背景：** 19世纪有"琥珀色ukha"（янтарная уха）的说法——用藏红花（来自里海贸易）染出金黄色。富人家可能用香槟替代部分水来煮呈宴客版。这道汤是复活节斋戒期结束后的传统第一餐。伏尔加河的渔夫用篝火上的吊锅煮ukha，这一画面被众多19世纪俄罗斯画家（如列宾）描绘。

### 4. Солянка（Solyanka）—— 酸辣杂烩汤

| 项目 | 内容 |
|------|------|
| **类型** | 浓厚酸辣汤 |
| **社会阶级** | 主要为城市中产及以上 |
| **起源** | 17–18世纪莫斯科地区，原称"селянка"（农家汤） |
| **主要原料** | 多种肉类（牛肉、火腿、香肠、肾脏）、腌黄瓜、腌黄瓜卤水（рассол）、橄榄、刺山柑（каперсы）、柠檬片、番茄酱、酸奶油 |

**做法概要：**
- 将至少三四种肉类（熏肉、煮牛肉、小牛肉肾、火腿/香肠）切成小丁或短条。
- 腌黄瓜切片或切丁，保留其酸卤水。
- 洋葱炒至金黄，加入番茄酱（19世纪末开始有罐装番茄酱）。
- 所有材料入牛肉高汤煮20–30分钟。
- 上桌前加入黑橄榄、刺山柑、撒莳萝，每碗放一勺酸奶油和一薄片柠檬。

**历史背景：** Solyanka本是清理厨房边角料（剩肉和腌菜）的实用料理——这也是其名字的由来（"соль"=盐，泛指各种咸味食材）。19世纪中后期从农家菜升级为小酒馆甚至贵族宴席的热门汤品。其浓烈的酸、咸、辣复合味非常配伏特加，因此在zakuski桌上极受欢迎。

### 5. Рассольник（Rassolnik）—— 腌黄瓜米汤

| 项目 | 内容 |
|------|------|
| **类型** | 咸酸清汤 |
| **社会阶级** | 城市中产及乡村 |
| **起源** | 18世纪末–19世纪初 |
| **主要原料** | 腌黄瓜（солёные огурцы）、腌黄瓜卤水、珍珠大麦（перловка）或大米、牛/鸡肾、土豆、胡萝卜、洋葱 |

**做法概要：**
- 珍珠大麦（或大米）事先浸泡。
- 肾脏焯水去腥后切丁，与蔬菜同炒，加腌黄瓜丁和卤水。
- 所有材料入高汤熬煮至大麦软烂。
- 出锅前可加一点酸奶油。

**历史背景：** Rassolnik从solyanka中分化而来，特点是必须含腌黄瓜卤水（рассол），且以谷物（通常是大麦或大米）为主，比solyanka清淡。19世纪中叶后，这道汤被收录进了Molokhovets的经典菜谱，成为俄国中产家庭餐桌上的常见汤品。

### 6. Окрошка（Okroshka）—— 冷汤

| 项目 | 内容 |
|------|------|
| **类型** | 夏日冷汤 |
| **社会阶级** | 全民 |
| **起源** | 伏尔加河流域，最早的冷汤形式可追溯到中世纪 |
| **主要原料** | 格瓦斯（квас）为汤底、煮熟的肉丁（牛肉/牛舌/鸡肉/火腿）、煮土豆、水煮蛋、小萝卜、鲜黄瓜、葱、莳萝、酸奶油、芥末 |

**做法概要：**
- 所有固体食材（肉、蛋、土豆、黄瓜、小萝卜）切成匀称小丁。
- 葱花加盐捣出汁，混入切碎的莳萝。
- 放入碗中，倒入冰镇的格瓦斯。
- 配酸奶油和俄式芥末（горчица）上桌。19世纪也有用开菲尔（кефир）替代kvass的变体。

**历史背景：** Okroshka是一道"不必开火"的夏日冷菜，名字来自"крошить"（切碎）。在炎热的大陆性气候夏季，这道菜既是汤又是主食。农民阶层的版本不加肉，纯素（仅蔬菜+格瓦斯）；贵族版的用料更为讲究。

### 7. 其他汤类速览
- **Гороховый суп（豌豆汤）**：干豌豆煮至软烂，配熏肉或火腿，是冬季日常浓汤。
- **Грибной суп（蘑菇汤）**：干牛肝菌或鲜蘑菇熬制，常加珍珠大麦，斋戒期的重要蛋白质来源。
- **Лапша（面条汤）**：自制鸡蛋面下入鸡汤或牛肉汤，最朴素的家常汤。
- **Ботвинья（Botvinya / 甜菜叶冷汤）**：用嫩甜菜叶、菠菜、酸模（щавель）煮出的绿色冷汤，配格瓦斯、冰块和鲟鱼肉/小龙虾尾，是19世纪贵族夏日的奢侈冷盘。

## 二、冷盘与开胃菜（Закуски）

Zakuski（закуски，字面意为"小咬一口"）是俄罗斯饮食文化中最具特色的部分。它不仅仅是一道道小菜，而是一整套**配有伏特加的冷餐台**传统。19世纪，这一传统发展至鼎盛——贵族府邸的zakuski桌可多达30–50种小碟，从名贵鱼子酱到平民腌黄瓜，琳琅满目。

### 8. Икра（Ikra / 鱼子酱）

| 类型 | 产区 | 特点 |
|------|------|------|
| **黑鱼子酱（Чёрная икра）** | 里海、伏尔加河下游、阿斯特拉罕 | 鲟鱼卵（белуга 大白鲟 / осётр 鲟鱼 / севрюга 闪光鲟），19世纪产量极大，价格相对亲民 |
| **红鱼子酱（Красная икра）** | 远东以及西伯利亚河流 | 鲑鱼卵（кета大马哈鱼/горбуша粉鲑），颜色橙红，价格便宜 |

**历史背景：** 19世纪，里海鲟鱼资源极为丰富，黑鱼子酱远非今天的天价奢侈品。一份好的zakuski桌上必然有鱼子酱——佐以切碎的煮蛋白、蛋黄、洋葱、柠檬汁和黄油，舀在小片黑面包或blini上。Molokhovets的书中甚至给出了如何用鱼子酱做冷肉冻（заливное）的菜谱。

### 9. 经典Zakuski菜品速览

| 菜名 | 描述 |
|------|------|
| **Селёдка под шубой** 的前身 | 腌鲱鱼配洋葱圈和煮土豆、撒葵花籽油和莳萝，是19世纪末出现的搭配（正式的"穿皮草大衣的鲱鱼"层叠沙拉是苏联时期的定型） |
| **Заливное（肉冻/鱼冻）** | 用浓肉汁（含明胶）浇在熟肉块/鱼肉/蔬菜上，冷却成冻。贵族的zаlivnoe常做成精美造型，中间嵌入煮鹌鹑蛋、小胡萝卜花等装饰 |
| **Холодец / Студень（肉冻）** | 农民版的zаlivnoe，用猪脚、猪耳朵、牛蹄长时间熬制出天然胶质，加上蒜末和黑胡椒，冷却后自然凝固 |
| **Солёные грибы（盐渍蘑菇）** | 秋季采摘的森林蘑菇（松乳菇рыжики、乳菇грузди、牛肝菌белые грибы等），用盐水+香料（莳萝花、蒜瓣、黑加仑叶）直接腌制，不加醋，是纯粹的乳酸发酵 |
| **Маринованные грибы（醋渍蘑菇）** | 用醋+香料的腌渍法，保存期更长 |
| **Солёные огурцы（腌黄瓜）** | 俄罗斯腌黄瓜不用醋，只用盐水和莳萝花、蒜、辣根叶（хрен）、橡树叶自然发酵 |
| **Квашеная капуста（酸渍卷心菜）** | 切丝卷心菜加盐和胡萝卜，自然乳酸发酵，有时加入越橘（брусника）或苹果增加甜酸层次 |
| **Мочёные яблоки（浸渍苹果）** | 安东诺夫卡品种的苹果浸泡在蜂蜜+盐+黑麦面粉的水中发酵，酸甜可口，冬季解腻 |
| **Студень из рыбы（鱼冻）** | 用鲟鱼头、梭鲈骨熬出胶质，加入大块鱼肉冷却成型 |
| **Хрен（辣根酱）** | 新鲜辣根磨碎，加少许醋和糖，配肉类zakuski |
| **Горчица（俄式芥末）** | 比法式芥末更辛辣，用芥末粉+热水+植物油+糖+盐调制 |

## 三、馅饼、馅包与鱼肉大馅饼（Пироги, Пирожки и Кулебяка）

俄语"пирог"（pirog，大型馅饼）和"пир"（盛宴）同源，揭示了馅饼在俄罗斯文化中的核心地位。19世纪是俄罗斯馅饼艺术的黄金时代。

### 10. Пироги（Pirogi）—— 俄罗斯传统大馅饼

| 项目 | 内容 |
|------|------|
| **类型** | 大型酵母发酵馅饼 |
| **社会阶级** | 全民（不同馅料区分阶级） |
| **面团** | 酵母发酵面团、千层酥皮（受法国影响）、或半发酵面团 |
| **馅料种类** | 荤馅（肉+洋葱、鱼肉+米饭+莳萝）、素馅（卷心菜+煮蛋+洋葱、蘑菇+荞麦粥、葱+蛋）、甜馅（浆果、苹果、奶渣+葡萄干） |

**做法概要：**
- 制作发酵面团（温水+酵母+面粉+鸡蛋+黄油），醒发两轮。
- 擀成约1cm厚的面皮，铺一层馅料，再盖一层面皮，边缘捏紧封口。表面刷蛋液。
- 烤箱中温（180–190°C）烤40–60分钟至金黄。
- 甜馅Pirog可撒糖粉。

**社会阶级差异：**
- 贵族：內馅可能是鲟鱼肉+米饭+莳萝，用千层酥皮包裹；或做成французский пирог（法式酥皮大馅饼）。
- 中产：羊肉+卷心菜、鸡蛋+大葱为常见馅料。
- 农民：荞麦粥+洋葱（最朴素）、卷心菜+碎蛋。

### 11. Пирожки（Pirozhki）—— 小馅包

| 项目 | 内容 |
|------|------|
| **类型** | 单手可持的小型馅包 |
| **社会阶级** | 全民 |
| **烹饪方式** | 烤（печёные）或炸（жареные） |
| **常见馅料** | 肉+洋葱、卷心菜+煮蛋、土豆泥+蘑菇+洋葱、米饭+蛋、肝泥（печень）、鱼肉、苹果/樱桃（甜馅）、奶渣（творог）+糖 |

**19世纪历史：** Pirozhki在19世纪因街头小贩和火车站的pirozhki摊位而走红全国。烤制的更接近家庭版本，油炸的在市集和街头最为常见。到了19世纪末，莫斯科和圣彼得堡出现了专门的"пирожковая"（pirozhki小店），出售多种馅料的现做小馅包。

### 12. Кулебяка（Kulebyaka）—— 多层鱼肉大馅饼

| 项目 | 内容 |
|------|------|
| **类型** | 大型长条多层馅饼 |
| **社会阶级** | 中产及以上 |
| **馅料分层** | 底层——煮熟的荞麦粥或米饭；中层——鱼肉（鲟鱼/梭鲈/三文鱼）；上层——炒蘑菇洋葱；每层之间用薄面饼（блинчики）隔开 |
| **特点** | 横截面能看到清晰的层次，形似长条形面包，是19世纪俄罗斯宴会桌的标志性大菜 |

**历史背景：** Kulebyaka的法语名"coulibiac"使其在19世纪末进入巴黎高级餐厅菜单，成为法式俄餐（cuisine russe à la française）的招牌菜。完美的kulebyaka要求切开后各层馅料不混、面皮酥脆、内馅湿润多汁。

### 13. Расстегай（Rastegai）—— 开口小馅饼

| 项目 | 内容 |
|------|------|
| **类型** | 顶部开口的船形馅饼 |
| **社会阶级** | 城市各阶级 |
| **特点** | 顶部留有一个小开口，吃之前在开口处浇入一勺融化的黄油或鱼汤，馅料通常为鱼肉+米饭+洋葱 |

**历史背景：** Rastegai的名字来自"расстёгивать"（解开扣子），据说因为太好吃了，吃之前要解开衣扣。19世纪莫斯科的小酒馆（трактир）常以此配ukha鱼汤一同售卖。

### 14. 其他馅饼速览
- **Ватрушка（Vatrushka）**：圆形开口小面饼，中间凹陷处填入甜奶渣（творог + 糖 + 鸡蛋 + 葡萄干），是茶点经典。
- **Курник（Kurnik / 鸡肉婚宴大馅饼）**：传统俄罗斯婚礼必备。多层馅料——鸡肉、蘑菇、米饭、鸡蛋——用薄面饼隔层，外面是厚厚的酥皮外壳。俄语谚语说"没有kurnik就不算婚礼"。
- **Чебурек（Cheburek）**：半月形油炸薄皮肉馅饼，源自克里米亚鞑靼人，19世纪在黑海沿岸和南俄城市流行。

## 四、煎饼类（Блины и Оладьи）

### 15. Блины（Blini）—— 俄罗斯薄煎饼

| 项目 | 内容 |
|------|------|
| **类型** | 发酵薄煎饼 |
| **社会阶级** | 全民 |
| **节日关联** | 谢肉节（Масленица / Maslenitsa，大斋前一周）的标志食品 |
| **面粉** | 传统用荞麦粉（гречневая мука），19世纪流行加入等量小麦粉以改善口感。酵母发酵版最为正宗 |

**做法概要（传统酵母版）：**
- 温水+酵母+荞麦粉+小麦粉调成稀面糊，发酵2–3小时，待表面出现气泡。
- 加入打发的蛋白、融化的黄油、少许盐和糖，再短暂二发。
- 用铸铁小煎锅（сковорода）抹猪油（сало）或黄油，舀一勺面糊摊薄。
- 两面煎至金黄。
- 叠成一摞，每层之间抹黄油。

**传统吃法：** 热blini搭配融化黄油、酸奶油（сметана）、鱼子酱（红或黑）、腌鲱鱼、烟熏三文鱼、蜂蜜、果酱（варенье）或融化的澄清黄油。谢肉节期间，人们一天可能吃几十张blini——圆形的金黄的blini象征太阳，庆祝冬去春来。

### 16. Оладьи（Oladyi）—— 厚煎饼

| 项目 | 内容 |
|------|------|
| **类型** | 小厚煎饼（类似美式pancake的大小和厚度） |
| **与Blini的区别** | Oladyi更厚、更蓬松，常用小苏打（19世纪引入）或酵母发泡；直径约6–8cm |
| **常见版本** | 苹果oladyi（面糊中加入苹果丁）、南瓜oladyi、奶渣oladyi（творожные оладьи） |

### 17. Блинчики（Blinchiki）—— 法式薄煎饼

| 项目 | 内容 |
|------|------|
| **类型** | 极薄的无酵母煎饼（类似法式crêpe） |
| **起源** | 受19世纪法餐影响 |
| **用途** | 通常卷馅食用：甜版卷奶渣+葡萄干或果酱；咸版卷肉碎+洋葱 |
| **社会阶级** | 主要在中产及以上家庭 |

## 五、肉类主菜（Мясные блюда）

### 18. Бефстроганов（Beef Stroganoff）—— 斯特罗加诺夫牛肉

| 项目 | 内容 |
|------|------|
| **类型** | 牛肉丝佐酸奶油酱 |
| **社会阶级** | 贵族 → 中产（19世纪末普及） |
| **诞生时间** | 19世纪中叶，最早见载于Molokhovets《给年轻主妇的礼物》1871年版 |
| **命名来源** | 以斯特罗加诺夫伯爵家族命名，据传是其法国厨师为年迈伯爵发明的嫩肉做法 |
| **原始配方** | 嫩牛肉切成小方丁（非长条！），用盐和五香粉 （allspice）干腌2小时，黄油煎至变色，加入少量高汤和酸奶油（сметана）略微加热即成，前后仅需一分钟 |

**演变：** Molokhovets的原版并没有蘑菇和洋葱（那是后来巴黎和美国的改良）。19世纪末的俄国版本通常是将牛肉条与芥末酱和酸奶油同炒，配炸土豆条或荞麦粥。

### 19. Пожарские котлеты（Pozharsky Cutlets）—— 波扎尔斯基肉饼

| 项目 | 内容 |
|------|------|
| **类型** | 裹面包屑炸制的鸡肉饼 |
| **社会阶级** | 贵族 → 全民 |
| **诞生时间** | 19世纪早期 |
| **传说背景** | 波扎尔斯基公爵要在托尔若克（Торжок）宴请沙皇亚历山大一世，原定的小牛肉没了，急中生智用鸡肉剁碎做成肉饼，裹面包屑黄油煎炸，沙皇大为赞赏 |
| **原料** | 鸡胸肉 + 黄油 + 奶油 + 白面包屑（内外均用） |

**19世纪版本：** 鸡肉剁成极细的肉泥，混入大量黄油和奶油（确保嫩滑），做成椭圆形肉饼，裹上碎面包屑，用澄清黄油（топлёное масло）煎至外酥内嫩。上桌时通常配奶油蘑菇酱。

### 20. Котлеты（Kotlety / 俄式肉饼）

| 项目 | 内容 |
|------|------|
| **类型** | 家常炸肉饼 |
| **来源** | 法式côtelette（排骨/肉片），18–19世纪传入后演变 |
| **19世纪版本** | 牛肉/猪肉/混合肉末 + 泡过牛奶的白面包（бyлка）+ 鸡蛋 + 洋葱，搅匀做成椭圆形，裹面包屑煎烤 |
| **社会阶级** | 全民（用料不同） |
| **贵族版** | 纯小牛肉或松鸡肉末，配蘑菇酱；**农民版**：含更多面包和土豆填充 |

### 21. 其他肉类主菜速览

| 菜名 | 描述 |
|------|------|
| **Жаркое（俄式烤肉/炖肉）** | 大块牛肉（或猪肉/羊肉）与土豆、胡萝卜、洋葱一起放在陶瓷罐（горшочек）中，盖紧后在俄式炉灶（русская печь）中慢炖数小时，肉质酥烂，汤汁浓郁。这是最经典的俄式家庭周日菜 |
| **Телятина запечённая（烤小牛肉）** | 整块小牛肉抹香料后烤制，19世纪贵族府的常见大菜 |
| **Баранина с кашей（羊肉配荞麦粥）** | 羊肉块与荞麦粥同煮，类似中亚plov，在顿河哥萨克和南俄地区极流行 |
| **Свинина тушёная с капустой（猪肉炖酸菜）** | 农民阶层冬季基本菜，肥猪肉与酸菜一块在炉灶中慢炖 |
| **Гуляш（гусаш / Goulash）** | 源自匈牙利-奥地利帝国，19世纪下半叶传入俄国的牛肉辣椒炖菜 |
| **Язык отварной（煮牛舌）** | 整条牛舌慢煮至嫩，去皮切片，配辣根酱醋汁，是zakuski桌上的经典冷盘 |
| **Почки по-русски（俄式炒腰花）** | 牛腰切花，与腌黄瓜、洋葱同炒，加酸奶油和白兰地，是19世纪小酒馆的热门下酒菜 |

## 六、饺子类（Пельмени и Вареники）

### 22. Пельмени（Pelmeni）—— 西伯利亚肉饺

| 项目 | 内容 |
|------|------|
| **类型** | 小号水饺（直径约3–4cm），面皮极薄（约1mm），圆皮对折捏成耳形 |
| **起源** | 西伯利亚原住民（科米人、乌德穆尔特人），"пельмень"源自科米语"пель-нянь"= 面包耳朵 |
| **传播** | 16–18世纪沿西伯利亚商路传入乌拉尔和莫斯科，到19世纪中后期成为全俄皆知的食物 |
| **馅料** | 牛肉+猪肉+羊肉混合肉末（经典三分肥比）、洋葱碎、黑胡椒、盐。传统上不加蒜、不加蔬菜、不加米饭 |
| **制作** | 西伯利亚传统是全家围坐大批量制作，冬季在户外直接冷冻（天然冰箱），冻硬的pelmeni装袋储存，吃时直接下沸水煮 |

**吃法：** 煮熟后盛入深盘，放一大块黄油，配酸奶油（сметана）或醋，撒黑胡椒。也有人蘸芥末或辣根酱。19世纪西伯利亚猎人携带冻pelmeni出发，在野外可直接煮食——被誉为"最早的速冻食品"。

### 23. Вареники（Vareniki）—— 乌克兰风格素/甜馅饺子

| 项目 | 内容 |
|------|------|
| **类型** | 半月形大饺子（个头比pelmeni大） |
| **起源** | 乌克兰，18–19世纪传遍南俄 |
| **馅料** | 酸樱桃（вишня）、奶渣（творог）+糖、土豆泥+炒洋葱+蘑菇、卷心菜 |
| **与Pelmeni区别** | Vareniki多甜馅或素馅，肉馅少见；皮略厚。甜版吃时加酸奶油和糖，或淋蜂蜜 |

### 24. 其他面饺类速览
- **Манты（Manty）**：中亚/西伯利亚大蒸饺，羊肉+洋葱+肥肉丁馅，用蒸笼蒸制，配酸奶油或辣油醋汁
- **Ушки（Ushki / "小耳朵"）**：微型饺子，内含蘑菇泥和洋葱碎，放入shchi白菜汤或borscht中同食
- **Кундюмы（Kundyumy）**：古罗斯修道院蘑菇烤饺——斋戒期的饺子，用蘑菇泥做馅，先烤后文火炖煮

## 七、鱼类与海鲜（Рыба и морепродукты）

俄罗斯有漫长的河流和海岸线，鱼类在餐桌上的地位不亚于肉类。尤其在东正教斋戒期，鱼在某些日子是被允许的（如圣母领报节和圣枝主日），因此鱼类菜肴异常丰富。

### 25. Осетрина / Белуга / Стерлядь（鲟鱼类）

| 鱼种 | 特点 |
|------|------|
| **Осётр（普通鲟鱼）** | 伏尔加河、里海主产，19世纪产量巨大 |
| **Белуга（大白鲟）** | 最大可达数吨，其鱼子酱最为名贵 |
| **Стерлядь（小体鲟）** | 较小（约5–10kg），肉质极嫩，"стерляжья уха"（小体鲟鱼汤）是极品 |

**常见做法：**
- **整条烤鲟鱼**：腹中填满香草和柠檬片，刷黄油烤制，上桌时配柠檬和热黄油
- **Заливное из осетрины（鲟鱼冻）**：鲟鱼煮熟切成大块，用鱼骨熬出的浓汤浇淋，冷却成琥珀色剔透鱼冻
- **鲟鱼肉排**：切成厚排，裹面包屑煎
- **Кольцо из стерляди（环形清蒸小体鲟）**：将整条小体鲟盘成环状，放入香槟蒸汽中清蒸，是19世纪圣彼得堡顶级法餐厅的招牌菜

### 26. 其他鱼类海鲜速览

| 菜名 | 描述 |
|------|------|
| **Судак отварной（煮梭鲈）** | 整条梭鲈（судак，伏尔加河的标志性白肉鱼）香料高汤煮熟，上桌配波兰酱（黄油+碎煮蛋+面包屑）或荷兰酱 |
| **Щука фаршированная（填馅梭子鱼）** | 整条梭子鱼剥下完整鱼皮，鱼肉去刺后与白面包、洋葱、鸡蛋搅成鱼泥，填回鱼皮中，还原成鱼形，蒸或烤制。犹太裔俄国人也做填馅鲤鱼（гефилте фиш） |
| **Форель по-русски（俄式鳟鱼）** | 鳟鱼用酸奶油、蘑菇、洋葱焗烤 |
| **Селёдка（腌鲱鱼）** | 大西洋和波罗的海鲱鱼（сельдь），盐腌后是全民最便宜的蛋白质来源。配煮土豆、洋葱圈、黑面包、葵花籽油 |
| **Копчёная рыба（烟熏鱼）** | 鲟鱼、鲑鱼、鳟鱼、鳗鱼热烟熏或冷烟熏，是zakuski桌的重要构成 |
| **Раки варёные（煮小龙虾）** | 淡水小龙虾（раки），用莳萝和盐煮，佐啤酒。19世纪俄罗斯河流中小龙虾极多，是平民美食 |
| **Угорь копчёный（烟熏鳗鱼）** | 波罗的海地区特产 |

### 27. Тельное（Telnoye / 鱼肉糜料理）

| 项目 | 内容 |
|------|------|
| **类型** | 用鱼肉泥（梭鲈或鲟鱼）做成的各种造型菜 |
| **历史** | 起源于17世纪的修道院斋戒饮食，19世纪进入贵族菜谱 |
| **经典菜式** | 鱼肉泥加入浸过牛奶的白面包、鸡蛋、奶油，做成半月形肉饼（тельное из судака），裹面包屑炸至金黄，外形模仿котлеты（肉饼），内部雪白鲜嫩 |

## 八、粥与谷物（Каши）

俄语"каша"（kasha）泛指一切用谷物熬煮的食物。俄罗斯谚语"щи да каша — пища наша"（白菜汤和粥是我们的食粮）尽显其根基地位。19世纪，粥既是最贫穷者的每日主食，也可以是贵族宴席上精心烹制的甜品。

### 28. 主要粥品

| 粥名 | 谷物 | 做法/特点 | 社会阶级 |
|------|------|-----------|:--:|
| **Гречневая каша（荞麦粥）** | 荞麦米（гречка） | 俄罗斯iconic粥。水或牛奶煮至软糯，加黄油。也可炒香后再煮（更香浓） | 全民 |
| **Пшённая каша（小米/黄米粥）** | 小米/稷米（пшено） | 最古老的俄罗斯粥之一，牛奶煮，加蜂蜜或黄油 | 全民（农民尤多） |
| **Манная каша（碎麦粥）** | 粗粒麦粉/西米（манка） | 用牛奶煮成，加糖和黄油，儿童和病人的食物 | 全民 |
| **Перловая каша（珍珠大麦粥）** | 珍珠大麦（перловка） | 嚼劲十足，常配蘑菇或肉 | 全民 |
| **Рисовая каша（大米粥）** | 大米 | 受中亚影响，19世纪渐普及，牛奶煮甜版或水煮咸版 | 中产以上 |
| **Овсяная каша（燕麦粥）** | 燕麦 | 尤其在北方和波罗的海地区 | 全民 |
| **Полбенная каша（斯佩耳特小麦粥）** | 斯佩耳特小麦 | 古罗斯古老品种，口感独特 | 农民为主 |

**食用习惯：** 传统俄式炉灶（русская печь）的余温适合慢煮粥——将谷物料、水/牛奶和盐放入陶罐（горшочек），封口后放在炉灶中过夜，次日早上取出，粥已焖得软糯浓香。

### 29. Гурьевская каша（Guriev Kasha）—— 古里耶夫甜品粥

| 项目 | 内容 |
|------|------|
| **类型** | 烤制甜点粥 |
| **社会阶级** | 贵族 |
| **诞生时间** | 19世纪初 |
| **命名来源** | 据传为财政大臣古里耶夫伯爵（граф Дмитрий Гурьев）的厨师所创，1820年代流行 |
| **做法** | 碎麦粥（манка）用牛奶煮成厚糊状，倒入浅烤盘，上面产生一层牛奶皮（пенки）后揭下备用。将粥与核桃碎、杏仁、蜜饯水果（цукаты）、香草层层交替铺叠，最上一层撒糖粉，入烤箱烤至表面焦糖化 |
| **特点** | 19世纪奢华到可以替代冰淇淋上桌。冷却后切块食用，内有层层牛奶皮（молочные пенки），口感丰富。是果戈理在《死魂灵》中提到过的名菜 |

### 30. 其他谷物速览
- **Семенуха / Сименуха**：19世纪流行的农家荞麦粥吃法——炒洋葱+炒蘑菇+碎煮鸡蛋拌入热荞麦粥
- **Драгомировская каша（德拉戈米罗夫粥）**：献给沙皇的荞麦粥奢华版，加上等牛骨髓和松露
- **Кутья（кутя / Kutya）**：葬礼和追思餐的仪式性甜麦粥——用煮熟的小麦或大米，加蜂蜜、罂粟籽（мак）、坚果碎和葡萄干，象征复活与永生

## 九、蔬菜、蘑菇与腌渍食品（Овощи, Грибы и Соленья）

俄罗斯的严冬使得蔬菜的新鲜食用期极短，因此腌渍（соление）、发酵（квашение）、浸渍（мочение）和风干（сушка）成为核心的保存智慧。这些技术不仅是实用手段，更演化出丰富的美味谱系。

### 31. 核心腌渍品

| 腌渍品 | 方法 | 关键点 |
|------|------|------|
| **Солёные огурцы（腌黄瓜）** | 盐水自然发酵法 | 黄瓜+水+盐+莳萝花伞（зонтики укропа）+蒜瓣+辣根叶+黑加仑叶+橡树叶。放入橡木桶中，压重石，地下室发酵2–4周。无醋、无糖，纯乳酸发酵。成品酸脆且带有莳萝和香叶的复合香气 |
| **Квашеная капуста（酸卷心菜）** | 切丝揉盐后自发酵 | 卷心菜切细丝，加盐揉搓出汁，加胡萝卜丝，有时加入整颗小苹果或越橘。压紧封口，室温发酵数日。酸香爽脆，富含维生素C，是预防坏血病的冬季关键食品 |
| **Мочёные яблоки（浸渍苹果）** | 蜂蜜盐水低温浸渍 | 安东诺夫卡苹果（антоновка）+ 水 + 蜂蜜 + 盐 + 黑麦面粉或麦芽，放入桶中浸泡30–40天。成品酸甜清爽，是冬季餐桌的解腻珍品 |
| **Солёные грибы（盐渍蘑菇）** | 干腌法（冷水/热水） | 俄罗斯秋季森林盛产可食用菌——松乳菇（рыжики）、乳菇（грузди）、卷边菇（волнушки）等。热水焯后分层码入桶中，每层撒盐+莳萝花+蒜+黑加仑叶，压重石。松乳菇3–5天即可吃，乳菇需40天 |
| **Солёные помидоры（腌番茄）** | 同腌黄瓜法 | 19世纪中后期番茄在俄国普及后开始腌渍 |

### 32. 日常蔬菜料理

| 菜名 | 描述 |
|------|------|
| **Картофель отварной с укропом（煮土豆配莳萝）** | 新土豆带皮煮，趁热剥皮，浇融化的黄油，撒粗盐和大量新鲜莳萝碎。这是19世纪最简朴也最广泛的家常配菜。土豆在俄国全面普及是19世纪中叶以后（之前被视为"魔鬼的果子"遭到过农民抵制） |
| **Картофельное пюре（土豆泥）** | 法国烹饪影响下的精致做法，加黄油和热牛奶捣成泥 |
| **Тушёная капуста（焖卷心菜）** | 洋葱和胡萝卜炒香后加入切碎的卷心菜慢炖，有时加番茄或酸菜 |
| **Винегрет из свёклы（甜菜根沙拉的前身）** | 煮甜菜根切丁，配葵花籽油、醋和腌黄瓜丁 |
| **Репа пареная（蒸芜菁）** | 土豆普及之前（18世纪末以前），芜菁是俄国的最主要蔬菜。将芜菁放入俄式炉灶中盖紧罐子蒸至软甜，至今仍是怀旧家常菜 |
| **Тыква запечённая（烤南瓜）** | 南瓜切块入炉灶慢烤，撒糖或蜂蜜 |

## 十、奶制品与蛋类料理（Молочные продукты и яйца）

乳制品在俄罗斯饮食中无处不在，尤其是发酵乳制品——它们既是食材，也是调味品，更是饮品。19世纪的俄国还没有现代冰箱，因此牛奶的"自然发酵"是其主要的保存形式。

### 33. 核心奶制品

| 奶制品 | 描述 |
|------|------|
| **Сметана（Smetana / 酸奶油）** | 俄餐的灵魂调味品——浓稠的酸奶油淋在汤（shchi, borscht）、饺子（pelmeni, vareniki）、煎饼（blini）、肉冻上。做法是将牛奶表层的奶油（сливки）撇下，自然发酵变酸变稠 |
| **Творог（Tvorog / 奶渣/农家奶酪）** | 介于奶酪和酸奶之间的产品——将酸牛奶（простокваша）加热后分离乳清，用布袋悬挂沥干。19世纪大量用于馅料（ватрушка, vareniki）、甜品（паска）、和日常早餐 |
| **Простокваша（Prostokvasha / 酸牛奶）** | 最原始的发酵乳——生牛奶在室温下自然发酵变稠变酸即成。有时加入黑面包皮帮助发酵。19世纪俄国农村家家户户自制 |
| **Ряженка（Ryazhenka / 烤酸牛奶）** | 将牛奶先在炉灶中慢火煨煮至微焦（产生焦糖色和坚果香），再冷却后加入酸奶油菌种发酵而成。色泽浅棕，风味独特，乌克兰/南俄起源 |
| **Варенец（Varenets / 烤酸奶）** | 类似ряженка，但只用烤过的牛奶自然发酵，不以菌种引导 |
| **Топлёное масло（澄清黄油）** | 将黄油慢火熬煮，蒸发水分和去除乳固体，留下纯黄油脂肪。烟点高，不易变质，是19世纪俄罗斯的主要烹饪油脂 |
| **Кефир（Kefir / 开菲尔）** | 源自高加索山区的发酵乳，19世纪下半叶传入广大的俄国，被梅奇尼科夫（诺贝尔奖得主）盛赞其益生功效 |

### 34. 蛋类料理

| 菜名 | 描述 |
|------|------|
| **Яйца с икрой（鱼子酱配蛋）** | 煮蛋切半，蛋黄取出与黄油和柠檬汁拌匀填回，表面铺一撮黑鱼子酱 |
| **Фаршированные яйца（酿蛋）** | 煮蛋切半，蛋黄与蘑菇泥、炒洋葱或腌鲱鱼碎混合后填回 |
| **Яичница-глазунья（煎太阳蛋）** | 黄油煎蛋，配黑面包和盐渍肥猪肉（сало） |
| **Яйца в смятку / крутую（溏心蛋/全熟蛋）** | 简单煮蛋配盐和黑面包，是农村和工人的常见早餐 |

## 十一、沙拉（Салаты）

沙拉在俄罗斯出现较晚，主要是19世纪法国烹饪影响的产物。但一经传入，俄国人便做出了自己独特的贡献。

### 35. Салат Оливье（Olivier Salad / 奥利维尔沙拉）

| 项目 | 内容 |
|------|------|
| **类型** | 蛋黄酱拌丁沙拉 |
| **诞生时间** | 1860年代 |
| **发明者** | 吕西安·奥利维尔（Люсьен Оливье），莫斯科艾尔米塔什餐厅（Эрмитаж）的比利时裔法国主厨 |
| **原始配方（1860s）** | 松鸡肉、牛舌、小龙虾尾（раковые шейки）、鱼子酱、刺山柑、腌黄瓜、煮土豆、煮鸡蛋，全部切小丁，淋上独家秘制普罗旺斯蛋黄酱（含法国醋、芥末、橄榄油、蛋黄）。这是一道奢侈到极致的餐厅菜，不是家常沙拉 |
| **秘密** | 奥利维尔将酱汁的配方带进了坟墓。他的招牌沙拉在欧洲贵族圈中成名，吸引了无数模仿者，但无人能完美复制 |
| **演变** | 到了19世纪末，简化的家常版（不含松鸡和鱼子酱）已经开始在莫斯科中产家庭出现，这为后来苏联时期的简化版（加煮香肠和青豆）奠定了基础 |

### 36. 其他沙拉
- **Винегрет（Vinegret / 甜菜根沙拉）**：煮甜菜根丁 + 土豆丁 + 胡萝卜丁 + 腌黄瓜丁 + 酸菜 + 洋葱，用葵花籽油和醋拌匀。19世纪下层和中产餐桌的常见冬季沙拉
- **Салат из свежей капусты（新鲜卷心菜沙拉）**：新鲜白卷心菜切丝 + 盐 + 葵花籽油 + 少许醋糖。最朴素的夏日农家沙拉
- **Мясной салат（肉沙拉）**：剩余烤肉切条，配腌黄瓜、洋葱、油醋汁

## 十二、面包与焙烤食品（Хлеб и выпечка）

谚语 "Хлеб — всему голова"（面包为万事之首）精确反映了面包在俄罗斯文化中的神圣地位。

### 37. 主要面包种类

| 面包 | 描述 |
|------|------|
| **Чёрный хлеб（黑面包 / 黑麦面包）** | 俄罗斯的国民面包。用黑麦面粉（ржаная мука）+ 麦芽（солод）+ 天然酵母（закваска，俗称"стартер"）长时间发酵制作。成品呈深棕近黑色，微酸，紧密厚实，可保存1–2周。19世纪，农民每日口粮2–3磅黑面包。两种著名品种：**Бородинский**（博罗季诺黑面包，含香菜籽和麦芽，据传是博罗季诺修道院的修女发明）和**Дарницкий** |
| **Серый хлеб（灰面包）** | 黑麦粉+小麦粉混合面包，介于黑面包和白面包之间 |
| **Белый хлеб（白面包）** | 用精白小麦粉制作，19世纪主要是城市中产和贵族的食物。农民只在节日才吃白面包。质量上乘的白面包称为**Ситный**（筛过的面粉所做） |
| **Калач（Kalach）** | 俄罗斯最古老的白面包之一。形状像带把手的锁，粗细两端口感不同——粗端松软，细端（"把手"）紧实有嚼劲。19世纪莫斯科的kalach名扬全国，被称为"莫斯科kalach"。谚语"калачом не заманишь"（用kalach也引诱不了他）= 怎么都劝不动 |

### 38. 烘焙食品

| 名称 | 描述 |
|------|------|
| **Бублик / Баранка / Сушка（面包圈三兄弟）** | 三种不同程度的面包圈——бублик最大最软（直径10–15cm）、баранка中等、сушка最小最硬。均用烫面（先沸水焯后烤制）制作。19世纪是非常流行的茶点零食 |
| **Булочка（小面包）** | 柔软的白面包卷，早餐配黄油和果酱，城市面包房的常见产品 |
| **Сухари（烤面包干）** | 将黑面包或白面包切片烤至干脆，可长期保存。浸入茶中食用，也是旅行和军粮的重要组成 |
| **Пряник（Pryanik / 俄式香料姜饼）** | 用蜂蜜、香料（肉桂、豆蔻、丁香、姜）、黑麦粉或小麦粉制作的糕点。19世纪形成了图拉（Тула）、维亚兹马（Вязьма）、戈罗杰茨（Городец）等著名产区。**Тульский пряник（图拉姜饼）** 内含果酱（сливовое повидло 李子酱为正宗），表面压印精美图案或文字，是旅行者必买的伴手礼 |
| **Коврижка（Kovrizhka / 蜂蜜香料糕）** | 类似prianik但更大更厚，通常切成方形或菱形，表面浇蜂蜜糖霜 |
| **Крендель（Krendel / 俄式扭结面包）** | 源自德国的Brezel（扭结面包），但更甜、更软，常做成8字形，撒粗糖粒 |

## 十三、甜点与糖果（Десерты и сладости）

19世纪俄国的甜品世界受到三条线索影响：古老的蜂蜜+浆果传统、法式糕点技艺的传入、以及俄国本土食材（如奶渣、荞麦粉、酸奶油）的创造性运用。

### 39. 经典甜点

| 甜点名 | 描述 |
|------|------|
| **Кисель（Kisel / Kissel / 水果淀粉冻）** | 俄罗斯最古老的甜点之一。远古版本用谷物（燕麦/小麦）发酵成酸味凝胶；19世纪出现了用浆果（蔓越莓клюква、树莓малина、醋栗смородина）果汁+土豆淀粉（картофельный крахмал）熬煮的现代版。成品从流质饮品到可用刀切的厚凝胶不等。液体版作饮料，固体版加牛奶或奶油作甜品 |
| **Варенье（Varenye / 整果果酱）** | 与西方搅碎的jam不同，俄式varenye保留完整果实形态，浸泡在透明糖浆中。材料包括草莓、樱桃、醋栗、黑加仑、杏、甚至玫瑰花瓣（варенье из роз）和核桃青果。是茶桌上的核心配角——直接从果酱碟中舀到茶匙上"咬"着吃（вприкуску） |
| **Пастила（Pastila / 俄式果糕）** | 用苹果泥（антоновка最正宗）+ 糖 + 打发的蛋白，搅打至蓬松后薄薄涂抹在布上，在微温的俄式炉灶中慢干数日，成品是松软的白色或浅棕色果糕片，可以卷起或多层压制。19世纪科洛姆纳（Коломна）和别廖夫（Белёв）的pastila名闻遐迩 |
| **Мороженое（冰淇淋）** | 18世纪末法式冰淇淋（glace）传入宫廷，19世纪中后期出现了用俄式农村冰冻技術制作的版本——"замороженное молоко"（冻牛奶），将加糖的浓奶油和果泥放入冰盐浴中搅打。古里耶夫粥冷切后也可替代冰淇淋 |
| **Хворост（Khvorost / "枯枝"）** | 薄面片切成菱形或长条（中间可划一刀翻出一头），猪油或黄油中炸至金黄酥脆，撒大量糖粉。名如其实——像一堆枯树枝。19世纪家庭和节庆极常见 |
| **Шарлотка（俄式苹果夏洛特）** | 受法式Charlotte借鉴而来，但19世纪的俄国版本变得更朴实：简化的海绵蛋糕面糊或白面包片，中间填入大量炖苹果 |
| **Медовик（Medovik / 蜂蜜千层蛋糕）** | 多层蜂蜜味薄蛋糕片夹酸奶油霜——据传是19世纪亚历山大一世皇后的厨师为其不爱吃蜂蜜的丈夫发明的"隐藏蜂蜜"蛋糕。多层面皮叠加酸奶油（сметана）+蜂蜜奶油夹心，冷藏一夜后面皮吸潮变软。原始版本远没有后来的10层那么多 |
| **Торт "Наполеон"（拿破仑蛋糕）** | 千层脆皮夹细腻的蛋奶酱（заварной крем），受法式mille-feuille启发，但俄式版本更加松脆且浸透了蛋奶酱。名字源于1812年卫国战争百年纪念（1912）——但蛋糕本身在19世纪下半叶已存在于圣彼得堡的法国面包房 |

### 40. 糖果和零食

| 名称 | 描述 |
|------|------|
| **Леденцы（硬糖）** | 糖浆熬至硬裂阶段，加入天然色素和香料，倒入模具成型。小熊形状的硬糖（мишка косолапый）是经典款 |
| **Семечки（葵花籽）** | 炒葵花籽，19世纪后期成为全俄国民级零嘴——街头、市场、谈话中无处不在地磕瓜子 |
| **Орехи в сахаре（糖衣坚果）** | 核桃或杏仁裹糖浆，节日甜盘必备 |
| **Мармелад（水果软糖）** | 19世纪主要用苹果胶制作，非明胶 |
| **Шоколад（巧克力）** | 19世纪下半叶进入俄国。最初是昂贵的热巧克力饮，后来才有固体巧克力。至19世纪末，圣彼得堡和莫斯科已有多家巧克力工厂 |

## 十四、饮品（Напитки）

### 41. 非酒精饮品

| 饮品 | 描述 |
|------|------|
| **Чай（茶）** | 中俄茶叶贸易始于17世纪，但19世纪是茶真正"征服"俄国的世纪。西伯利亚路线（чайный путь）的驼帮将中国茶砖从恰克图（Кяхта）运往莫斯科。到19世纪末，茶已成为从贵族到农民的全民饮品。喝茶的核心道具是**самовар（茶炊）**——金属壶中烧炭保持水温，顶部置茶壶（заварник）以濃茶（заварка）兑热水（кипяток）饮用。通常配果酱（вприкуску——将茶匙上的果酱直接送入口中再喝茶）、糖块（咬一块糖喝茶）、柠檬片、小面包圈（баранки/сушки）或prianiki（姜饼） |
| **Сбитень（Sbiten）** | 蜂蜜+水+香料（肉桂、丁香、姜、豆蔻、月桂）熬煮的热饮，是茶普及之前俄国最普遍的街头热饮。19世纪茶盛行后衰落，但仍有人售卖，尤其是冬日的市集。可以不加酒精，也可以加入少量红酒或白兰地 |
| **Квас（Kvass / 格瓦斯）** | 用黑麦面包或大麦+酵母+糖+水发酵而成的微碳酸饮料，酒精含量很低（0.5–2%）。有酸面包风味，俄国的国民饮料。19世纪还有水果kvass（梨、苹果、蔓越莓）、蜂蜜kvass（медовый квас）等多种变体。夏天街头到处是kvass桶车 |
| **Морс（Mors / 浆果汁饮料）** | 蔓越莓（клюква）或越橘（брусника）等野生浆果加水煮出汁，过滤后加糖/蜂蜜。是夏季常见消暑饮品 |
| **Взвар / Узвар（Vzvar / Uzvar）** | 干果（苹果干、梨干、李子干）加水慢火煮出的蜜饯果茶，圣诞晚餐的传统饮料 |
| **Берёзовый сок（白桦树汁）** | 春季从白桦树中采集的树汁，微甜清爽，可鲜饮或发酵。传统上认为有健康和养颜功效 |
| **Кофе（咖啡）** | 18世纪末–19世纪进入俄罗斯贵族圈，但不如茶普及。普通农民几乎不喝咖啡，因其昂贵 |
| **Цикорий（菊苣咖啡）** | 烤菊苣根磨粉冲水，作咖啡的廉价替代品，19世纪在低收入人群中颇为流行 |

### 42. 酒精饮品

| 饮品 | 描述 |
|------|------|
| **Водка（Vodka / 伏特加）** | 俄罗斯的灵魂之酒。19世纪的伏特加通常以黑麦或小麦为原料，蒸馏后经木炭过滤（применение угольной фильтрации — 这一经典的"莫斯科伏特加"风格确立于19世纪中后期）。19世纪末国家开始实行伏特加专卖制度（1894年，财政大臣维特推行）。酒精度通常在40%左右。喝法：冰镇后一口干，配zakuski（腌黄瓜一口、鱼子酱一口、鲱鱼一口），"без закуски водка не пьётся"（没有下酒菜就不喝伏特加） |
| **Хлебное вино（面包酒）** | 伏特加的前身，泛指黑麦/小麦蒸馏酒。19世纪上半叶仍常用此名称 |
| **Наливка（Nalivka）** | 用浆果（樱桃、黑加仑、树莓）浸泡在伏特加或белое вино中，加糖，密封数周。甜度高，是女士们喜爱的利口酒。与此区分的**настойка**（nastoyka）指用草药或香料浸泡的伏特加（如辣根伏特加хреновуха） |
| **Медовуха（Medovukha）** | 蜂蜜+水+酵母发酵的低度蜂蜜酒（2–12%），中世纪的国民酒，19世纪被伏特加和啤酒挤压但仍存 |
| **Пиво（啤酒）** | 主要在西部、波罗的海和城市酿造 |
| **Игристое вино / Шампанское（起泡酒）** | 19世纪俄国贵族宴会以法国香槟为标杆；克里米亚和阿布劳-久尔索（Абрау-Дюрсо）等地的俄国起泡酒产业也于19世纪末起步 |

## 十五、节日与宗教斋戒食品（Праздничная и постная кухня）

东正教对俄罗斯饮食的塑造不可估量：全年约180–200天为斋戒日（пост），在这些日子里，信徒不可食用任何动物来源的食品——肉、奶制品、蛋类、有时连鱼和油也在禁止之列。

### 43. 斋戒类型

| 斋戒期 | 时长 | 饮食限制 |
|------|------|------|
| **Великий пост（大斋）** | 复活节前7周 | 禁一切动物性食品（肉、奶、蛋）。某些日子连油（植物油除外）和鱼也不行 |
| **Рождественский пост（圣诞斋）** | 11月28日–1月6日 | 禁肉、奶、蛋；鱼在特定日期允许 |
| **Петров пост（彼得斋）** | 圣三一节后至6月29日 | 较宽松 |
| **Успенский пост（圣母安息斋）** | 8月14–27日 | 严格 |
| **每周三、五** | 全年 | 禁肉、奶、蛋（纪念犹大出卖和基督受难） |

**斋戒期的主力食品：** 蘑菇（干/鲜/腌渍）、黑面包、荞麦粥、小米粥、酸菜、腌黄瓜、各种浆果、坚果、蜂蜜。用葵花籽油或亚麻籽油替代黄油。修道院发展出了丰富的斋戒食谱——如蘑菇鱼子酱（грибная икра，剁碎的蘑菇炒洋葱替代真鱼子酱）、斋戒版shchi、斋戒版pirozhki等。

### 44. 节庆食品

#### 复活节（Пасха）

复活节是东正教最重要的节日，大斋结束后的饕餮盛宴。

| 食品 | 描述 |
|------|------|
| **Кулич（Kulich）** | 复活节圆柱形甜面包（形似主教冠/教堂圆顶）。丰富的配料：大量鸡蛋、黄油、糖、葡萄干、蜜饯果皮、杏仁、豆蔻或番红花。表面涂抹蛋白糖霜（белая глазурь），顶上撒彩色小米粒或装饰"ХВ"（Христос Воскресе / 基督复活）。通常在圣周六带到教堂接受神父祝福，周日早上全家分享 |
| **Пасха（Paskha / 奶渣蛋糕）** | 一种不用烘焙的甜奶渣点心。将奶渣（творог）+黄油+鸡蛋+酸奶油+糖+香草+葡萄干+杏仁拌匀，包入纱布，放入专门的木制梯形模具（пасочница，模具内壁刻有"ХВ"字样和十字架/花卉等浮雕图案），重物压迫过夜使多余乳清排出。出模后呈截顶金字塔形，象征耶稣的坟墓和复活 |
| **Крашеные яйца（彩蛋）** | 复活节煮蛋染成红色（洋葱皮是最传统的天然染料），象征基督的血和复活。富裕人家也有木制/瓷制/宝石制彩蛋（法贝热彩蛋于1885年首次为沙皇制作） |

#### 圣诞节（Рождество）

| 食品 | 描述 |
|------|------|
| **Сочиво / Кутья（Sochivo / Kutya）** | 圣诞前夜（Сочельник，1月6日）的仪式性甜粥，标志着圣诞斋的结束。小麦/大米+蜂蜜+罂粟籽+坚果碎+干果。全家在出现第一颗星后才开始吃 |
| **Гусь с яблоками（苹果烤鹅）** | 整鹅腹中填入苹果和酸菜烤制，是圣诞主菜 |
| **Взвар（Uzvar / 干果蜜饯）** | 圣诞晚餐的干果煮甜汤，配kutya食用 |
| **Пряники（Prianiki香料姜饼）** | 圣诞期间最受欢迎的甜点之一 |

#### 谢肉节（Масленица / Maslenitsa）

大斋前一周的谢肉节是"告别冬天、迎接春天"的狂欢——也是全年最盛大的blini煎饼节。这一周天天吃blini，配鱼子酱、酸奶油、黄油、腌鲱鱼、蜂蜜、果酱。最后一天（宽恕星期日）燃烧稻草人（чучело），象征告别冬天。

## 十六、街头食品与市集小吃（Уличная еда）

19世纪俄国城市和市集的街头食品文化极为丰富：

| 食品 | 描述 |
|------|------|
| **Пирожки（小馅包）** | 肉、卷心菜、土豆、蛋葱等馅料，街头小贩（pirozhnik / пирожник）挎篮或推车叫卖 |
| **Блины с пылу с жару（热煎饼）** | 街头煎饼摊位现场制作，热腾腾出锅即食 |
| **Сбитень（热蜂蜜饮）** | 冬日街头，сбитенщик（sbiten小贩）背着铜制sbiten茶炊，用瓷杯售卖热气腾腾的甜辣蜂蜜水 |
| **Калачи и баранки（白面包圈）** | 成串挂在木棍上售卖 |
| **Гречневики（荞麦饼）** | 荞麦面团做成小饼烤制，价格便宜的街头早餐 |
| **Варёные раки（煮小龙虾）** | 街头小贩用大锅煮新鲜小龙虾，盐+莳萝调味 |
| **Мочёные яблоки / груши（浸渍苹果/梨）** | 从桶中取出售卖的小吃 |
| **Семечки（葵花籽）** | 炒瓜子，有摊位可买，是19世纪最普遍的街头零嘴 |

## 十七、野味与狩猎料理（Дичь и охотничья кухня）

俄罗斯广袤的森林和草原提供了丰富的野味资源：

| 肉源 | 常见做法 |
|------|------|
| **Медвежатина（熊肉）** | 西伯利亚传统，切块长期腌制后烤或炖。熊掌视为珍馐 |
| **Лосятина（驼鹿肉）** | 北方和西伯利亚的重要肉源，腌/熏/风干 |
| **Кабанина（野猪肉）** | 猎人常做烤肉或炖菜，肉味比家猪更浓 |
| **Рябчик（松鸡）** | 最受推崇的野禽，肉质白嫩。Olivier沙拉原版使用 |
| **Тетерев（黑琴鸡）** | 另一种优质野禽 |
| **Куропатка（山鹑/石鸡）** | 烤或炖 |
| **Перепела（鹌鹑）** | 填馅烤制或做成肉冻 |
| **Заяц（野兔）** | 用酸奶油+蘑菇+洋葱慢炖，或做成肉酱паштет |
| **Фазан（野鸡/雉）** | 19世纪已有猎场专门放养 |

## 附录：19世纪俄罗斯各阶级日常饮食速览

| 社会阶级 | 日常饮食概况 |
|:--|------|
| **皇室与最高贵族** | 法餐影响深刻——菜单以法文书写，主厨多为法国人或受法训的俄国人。全套欧式晚宴：汤（consommé）+ 鱼（鲟鱼冻/三文鱼）+ 肉（烤小牛肉或填馅野禽）+ 沙拉 + 甜点 + 水果 + 咖啡。zakuski桌极尽奢华（30–50道冷盘）。酒水：法国香槟、匈牙利托卡伊、法国/克里米亚葡萄酒、伏特加。一天可吃4–5餐 |
| **中产（官僚、商人、专业人士）** | 19世纪下半叶崛起的社会力量。Molokhovets的菜谱书正是为他们而写。一日三餐：早餐（茶+白面包+黄油+冷肉）、午餐（3–4道菜，汤+肉/鱼+甜点）、晚餐（茶+夹馅食品或剩菜）。招待客人时会做zakuski+iи和法式甜点。肉类消费约每周数次 |
| **城市工人/市民** | 黑面包为基础。早餐：茶+黑面包+黄油（有时配鲱鱼）。午餐：shchi白菜汤或kasha粥+黑面包。晚餐：剩菜或茶+面包。肉不常见——香肠碎或便宜的内脏。街头pirozhki是偶尔的改善。工资的40–60%用于食物 |
| **农村中农** | 主要靠自家生产。主食：黑面包、荞麦粥、酸菜、腌黄瓜、土豆。汤（shchi）几乎每日出现。牛奶和奶制品自产。肉仅在节日（复活节、圣诞、村社守护圣人日）食用——通常宰一头猪/羊全村分享。鱼靠河流或池塘自捕。全年大量时间在遵守斋戒 |
| **最底层/农奴（1861年前）和失地农民** | 黑面包可能掺入麦麸/橡子粉/树皮粉。稀粥和菜汤为主。日常蛋白来源是腌鲱鱼（最便宜的鱼）。饥荒年份只能吃"饥饿面包"——加入藜麦种子（лебеда）、荞麦壳、甚至树皮粉末的粗劣面包。土豆是他们被地主强迫种植的作物，19世纪中期后才被广泛接受为食物 |

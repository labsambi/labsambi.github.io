# 20世纪90年代德国常见食物汇总

> 涵盖家常菜、街头小吃、餐馆宴饮、食谱与价格估算 | 时代坐标：两德统一后 | 货币：德国马克（DEM/DM），1995年约1£≈2.30 DEM，1 USD≈1.43 DEM，1€≈1.95583 DEM（1999年固定汇率）

---

## 一、时代背景：统一后的十年

20世纪90年代是德国饮食史上**东西融合与外部全球化冲击**的时期。

**核心事件**：
- **1990年两德统一**：东德人第一次可以走进西德超市——货架上的商品多样性是难以置信的奢侈。与此同时，东德的Trabant（产自Zwickau的特拉贝特车）排长队过境去买香蕉。东德品牌的食品（如Radeberger啤酒、Bautz'ner芥末、Spee洗衣粉）经历了一轮"东怀旧（Ostalgie）"——有的消失，有的留存。
- **Döner Kebab的全民化**：1972年Kadir Nurman在柏林发明了"德国版döner"（肉串烤→切片+面包+沙拉+酱汁）——在90年代成为**德国最大的快餐品类**。1995年柏林döner价格约4-6 DM。全德每天的döner消费量达300吨肉。
- **Currywurst文化战争**：虽然是1949年Herta Heuwer在柏林发明的，但咖喱香肠在90年代的德国饮食地图上已成为和啤酒一样的全民遗产。柏林甚至有一个"Currywurst Museum"（咖喱香肠博物馆）的计划在90年代末开始酝酿。
- **1996年疯牛病恐慌**：德国受到影响（相对英法较轻），有机农业市场（Bio）大幅增长。
- **Aldi的全球崛起**：Albrecht兄弟的Aldi在90年代已成为德国最大的食品零售商——"Aldisierung"（阿尔迪化）一词被用来描述德国饮食的标准化趋势。

**德国人的饮食节奏（约1995年）**：
- 6:30-7:30 Frühstück：面包（Brötchen/Graubrot）+ 黄油+ Wurst（冷香肠片）/Käse+ 果酱+ 煮蛋 — 德国的早餐在欧洲范围内是较丰盛的
- 9:30-10:00 Zweites Frühstück（第二早餐）——在学校和工作场所会有一个小面包或水果
- 12:00-13:00 Mittagessen（午餐——传统上一日主餐）
- 15:30-16:30 Kaffee und Kuchen（咖啡+蛋糕）——绝对不可侵犯的德国仪式！商店下午关门（Ladenschlussgesetz在90年代仍限制到晚6点和周六下午）
- 18:00-19:00 Abendbrot（晚间面包——传统上是冷食：面包+Wurst/Käse+沙拉）但90年代城市中产越来越多地改为热晚餐（warmes Abendessen）

## 二、日常家常菜（Hausmannskost）

### 2.1 德国厨房逻辑

- **Kartoffeln（马铃薯）是心脏**：盐煮土豆（Salzkartoffeln）、土豆泥（Kartoffelpüree）、煎土豆（Bratkartoffeln）、土豆丸子（Knödel/Klöße）、土豆饼（Reibekuchen）——德国菜中土豆的形态多到惊人。
- **Soße是灵魂**：酱汁（Bratensoße/肉汁、Senfsoße/芥末酱、Rahmsoße/奶油酱）是德国主菜不可妥协的组成部分。
- **地区性极强**：即使在统一后，德国饮食仍然极其地方性。巴伐利亚的Weißwurst和Brezn（白香肠+碱水面包）；施瓦本的Maultaschen和Spätzle；莱茵兰的Sauerbraten和Reibekuchen；北德的Grünkohl和Labskaus；柏林的Boulette和Currywurst——德国没有"德国菜"，只有"德国各地的菜"。

### 2.2 90年代西德城市家庭一周餐桌

| 星期 | 午餐（Mittagessen） | 晚餐（Abendessen） |
|:---:|------|------|
| 周一 | Nudeln mit Tomatensoße（番茄酱意面——90年代意大利风深入德国厨房） | Brotzeit（面包+Wurst+奶酪——经典冷晚餐） |
| 周二 | Bratwurst + Kartoffelpüree + Sauerkraut（煎香肠+土豆泥+酸菜） | 同上或剩菜 |
| 周三 | Geschnetzeltes mit Reis（奶油蘑菇猪肉条+米饭） | Salatteller（沙拉拼盘+面包） |
| 周四 | Frikadellen（肉丸）+ Kartoffelsalat（土豆沙拉）+ Senf（芥末） | Brotzeit |
| 周五 | Fischstäbchen（鱼条）+ Spinat（菠菜泥）+ Salzkartoffeln | Wurstbrot（香肠面包）或出去吃（Döner / Currywurst） |
| 周六 | Mittags可能外食——Imbiss oder Restaurant | 外食或自制——Raclette / 烧烤（夏天） |
| 周日 | **Sonntagsbraten**：猪颈/牛后腿烤肉（Rinder-/Schweinebraten） + Knödel（丸） + Rotkohl（红腌菜） + Soße（肉汁）——德国家庭周日餐桌的绝对仪式 | 冷食面包——不做饭 |

### 2.3 核心家常菜清单（16道）

| 序号 | 德语 | 中文 | 类型 | 说明 |
|:---:|------|------|:---:|------|
| 1 | **Sauerbraten** | 酸味炖牛肉 | 主菜 | 牛肉在醋+红酒+蔬菜中腌制数天→慢炖数小时——莱茵兰的国菜 |
| 2 | **Schweinebraten** | 烤猪肉 | 周日主菜 | 猪颈/肩慢烤至皮脆（Kruste是灵魂）→配Knödel+肉汁 |
| 3 | **Rinderrouladen** | 牛肉卷 | 主菜 | 薄牛肉片卷洋葱+培根+芥末+Gurke（腌黄瓜）→慢炖 |
| 4 | **Bratwurst / Rostbratwurst** | 煎香肠 | 主菜/快餐 | 各地不同——Nürnberger（小短香肠）、Thüringer（长炭烤香肠） |
| 5 | **Currywurst** | 咖喱香肠 | 快餐 | 炸/煎香肠切片→番茄咖喱酱+咖喱粉——柏林/鲁尔区之王 |

| 7 | **Kartoffelpuffer / Reibekuchen** | 土豆煎饼 | 街头/配菜 | 刨丝土豆+洋葱+蛋和少量面粉→油炸→苹果酱（Apfelmus） |
| 8 | **Schnitzel "Wiener Art"** | 维也纳式炸肉排 | 主菜 | 猪肉/小牛肉薄片三层面衣——配柠檬角和Preiselbeeren（越橘酱） |
| 9 | **Königsberger Klopse** | 柯尼斯堡肉丸 | 主菜 | 小牛肉/肉碎丸子在奶油刺山柑酱中——东普鲁士的遗产 |
| 10 | **Grünkohl mit Pinkel** | 羽衣甘蓝+香肠 | 北德冬季 | 慢炖数小时的羽衣甘蓝+烟熏香肠Pinkel+ Kassler（腌猪排）+ Salzkartoffeln |
| 11 | **Maultaschen** | 斯瓦本饺子 | 主菜/汤 | 大号ravioli——碎肉+菠菜+洋葱填馅——在肉汤里或煎着吃 |
| 12 | **Käsespätzle** | 奶酪面疙瘩 | 主菜/素食 | 自制Spätzle（软面疙瘩）+层层Bergkäse—烤至焦香+焦糖洋葱顶——德国最佳comfort food之一 |
| 13 | **Labskaus** | 海员杂碎 | 北德主菜 | 腌牛肉+土豆+洋葱+甜菜根——搅成粉红色浆→配煎蛋和rollmops（腌鲱鱼卷） |
| 14 | **Himmel und Erde** | "天堂与地" | 主菜 | 苹果泥（天堂）+土豆泥（地）+血肠（Blutwurst）+油煎洋葱——莱茵兰 |
| 15 | **Sauerkraut / Rotkohl** | 酸菜/红腌菜 | 配菜 | 德国餐盘的标志性配菜——酸菜（白）、红腌菜（苹果+丁香慢炖） |
| 16 | **Kartoffelsalat** | 土豆沙拉 | 配菜 | 南北分野——北方：蛋黄酱底、南方：醋+油+肉汤底（温热食用） |

## 三、街头小吃与Imbiss（小吃站）文化

### 3.1 德国街头小吃的核心品种

| 品种 | 说明 | 1995年价格（DEM） | 约合£ |
|------|------|:---:|:---:|
| **Döner Kebab** | 旋转烤肉切片→塞在Fladenbrot（扁面包）三角袋中+沙拉+酱汁——**90年代德国最大的快餐品类** | **4-6 DM** | £1.74-2.61 |
| **Currywurst** | 炸香肠切片→浇上番茄咖喱酱+咖喱粉→配Brötchen（小面包）或Pommes（薯条） | **3-4.50 DM** | £1.30-1.96 |
| **Bratwurst im Brötchen**（面包夹香肠） | 烤/煎香肠夹在小面包中+Senf（芥末）——Mittag（午餐）的简单解决 | **3-4 DM** | £1.30-1.74 |
| **Kartoffelpuffer / Reibekuchen** | 土豆饼+苹果酱——圣诞市场和人流多的地方 | **2-3 DM/个** | £0.87-1.30 |
| **Fischbrötchen**（鱼面包卷） | 北德特产：腌鲱鱼/烟熏鱼夹在面包卷里+洋葱——港口城市的标配 | **3-5 DM** | £1.30-2.17 |
| **Leberkäsesemmel** | 巴伐利亚面包卷夹厚片肝奶酪（Leberkäse，实际上没有肝没有奶酪——是精粉碎的猪肉+牛肉烤制）| **3-4 DM** | £1.30-1.74 |
| **Brezn**（碱水面包/Laugenbrezel） | 巴伐利亚的绝对象征 | **1-1.50 DM** | £0.43-0.65 |
| **Pommes**（薯条）+Mayo | 德国人发明了"薯条+蛋黄酱"这个组合——荷兰也说是他们发明的 | **2-3 DM**（小份） | £0.87-1.30 |

### 3.2 Imbiss（小吃站）和Biergarten（啤酒花园）

| 类型 | 说明 |
|------|------|
| **Imbissbude**（小吃亭） | 街角的标志性小铁皮亭——卖Currywurst/Bratwurst/Pommes/Bockwurst |
| **Dönerbude** | 每个街角至少一个——由土耳其裔德国人经营——90年代巅峰 |
| **Biergarten** | 巴伐利亚（尤其是慕尼黑）——在栗子树下，长木桌——自带食物（Brotzeit/冷盘面包）是可以接受的但饮料必须从garten买 |
| **Würstelstand** | 奥地利+巴伐利亚——夜间的香肠站——Bockwurst/Käsekrainer（奶酪香肠）/Currywurst——是夜里外出后的聚集点 |
| **Weihnachtsmarkt**（圣诞市场，11月末-12月） | Glühwein（热红酒）+ Bratwurst + Kartoffelpuffer + gebrannte Mandeln（焦糖杏仁）+ Lebkuchen（姜饼） |

### 4.3 东德-西德90年代的餐饮差异

| 维度 | 西德 | 东德（90年代新联邦州） |
|------|------|------|

| Döner分布 | 遍布 | 1990年后才开始出现 |
| 东德特有食物 | — | Ketwurst（类似于热的hot dog——将Bockwurst塞入面包卷中+番茄酱——GDR时期的街头食物；90年代仍在一些地方能买到）、Soljanka（苏式酸汤——番茄/蔬菜/香肠汤——来自苏联的遗产，东德人对其有怀旧情绪） |

## 五、特色饮品

### 5.1 啤酒——德国的液体面包

| 类型 | 代表品牌/地区 | 1995年价格 |
|------|------|:---:|
| **Pils**（淡色啤酒） | Bitburger、Krombacher、Warsteiner、Beck's | 2.5-4 DM（0.3L餐厅）/ 1-1.5 DM（0.5L瓶装超市） |
| **Weizen / Weißbier**（小麦啤酒） | Paulaner、Erdinger、Franziskaner | 3-4.5 DM（0.5L） |
| **Helles**（巴伐利亚清亮拉格） | Augustiner、Spaten | 3-4 DM（0.5L） |
| **Kölsch**（科隆特产） | 浅淡、顶部发酵、用细窄玻璃杯（Stange 0.2L）——Kölner在杯沿记号数量 | 1.5-2 DM/Stange |
| **Altbier**（杜塞尔多夫老式啤酒） | Diebels、Schlösser | 2.5-3.5 DM（0.25L） |
| **Maß**（1升大杯——巴伐利亚啤酒节） | 在Oktoberfest/Wiesn上按Maß出售 | 9-10 DM/Maß（1995年） |

**德国的啤酒纯净法（Reinheitsgebot）**：1516年以来只允许水+大麦+啤酒花的古老法律，在90年代仍然得到尊重和营销。

## 八、代表性食谱

### 8.1 Sauerbraten（莱茵兰酸味炖牛肉）

> 德国最著名的慢炖菜之一——酸腌数天后慢炖数小时。

**原料（6人份）**：
- Rindfleisch（牛肉——肩肉/sirloin/rump）：约1.5kg整块
- 腌制液：红酒醋（Rotweinessig）250ml + 干红葡萄酒500ml + 水500ml + 洋葱2个（切片）+ 胡萝卜2根+ 芹菜1块+ 胡椒粒10粒+ 丁香4粒+ 月桂叶3片+ Wacholderbeeren（杜松子）1小勺
- 煎肉用：Butterschmalz（澄清黄油——黄油提炼去水分和蛋白质）或植物油
- 炖汁用：葡萄干（Rosinen）一小把、蜂蜜/Pflaumenmus（李子酱）、Lebkuchen或Soßenlebkuchen（一种专门用于做酱汁的姜饼碎——这是Sauerbraten的独特"秘密"食材）
- 最后勾芡：淀粉
- 盐、糖

**做法**：
1. **前3-5天腌肉**：将腌制液所有材料煮沸后完全冷却（这步重要！）。将牛肉整块放入深容器中，加入冷却的腌制液——必须完全浸没肉。盖好。放入冰箱腌3-5天——每天翻一次面。
2. 取出肉，用厨房纸巾彻底擦干。过滤腌料，保留蔬菜和液体分别。
3. 煎锅加热Butterschmalz——中大火。将腌牛肉各面煎约10分钟至深棕色。
4. 取出肉。同锅炒腌料中的蔬菜（洋葱+胡萝卜+芹菜）约5分钟。
5. 将肉+蔬菜放到大炖锅中（或同锅）。加入一部分腌液至肉的3/4高度。
6. 加入葡萄干、Sauerbratengewürz（如果没单独备，可以用1小块肉桂+2粒丁香+胡椒粒用纱布包好）。
7. 煮开后盖盖转小火——炖至少2.5-3小时至肉完全软化。
8. 取出肉，保温。
9. **做酱汁**：过滤炖汁，倒回锅中。加入Soßenlebkuchen（或1大勺蜂蜜+半勺可可粉+少许肉桂和丁香——模拟Lebkuchen的味道）。如果太薄则用淀粉水勾芡。尝味：需要酸甜的平衡——可加糖/蜂蜜或更多醋。
10. 肉切片，将酱汁浇在上面。传统配：Kartoffelklöße（土豆丸，煮的）+ Rotkohl（红腌菜）或 Apfelrotkohl（苹果红腌菜）。

### 8.2 Kartoffelpuffer / Reibekuchen（土豆煎饼）

> 德国圣诞市场和街头最简单但也最迷人的味道——两个土豆+一个洋葱=奇迹。

**原料（约12-15个）**：
- 土豆（mehligkochend/粉质型最理想）：约1.5kg
- 洋葱：1个大的
- 鸡蛋：1个
- 面粉：2-3大勺
- 盐：1大勺
- Muskatnuss（肉豆蔻）：少许磨碎
- 植物油（油炸用）
- 苹果酱（Apfelmus）或糖粉作陪

**做法**：
1. 土豆去皮。**刨丝**（用Box Reibe/盒式刨丝器粗孔——或食品加工机的粗丝刀片）。
2. 将土豆丝放入一个大碗中——尽可能挤出多余的水分（或放入纱布/厨房毛巾中拧紧挤出液体）。保留挤出的液体在碗底沉淀——倒掉上清液后底层有土豆淀粉。
3. 洋葱也刨碎。加入土豆丝中。
4. 将保留的土豆淀粉加入混合物。加鸡蛋+面粉+盐+肉豆蔻。充分混合。
5. 大煎锅中加热足量的油（覆盖底面——约5mm深）到中高温。
6. 将一满大勺的土豆混合物放入热油中——立即用勺子压扁成圆饼形，约0.5cm厚。
7. 每面炸3-4分钟至深金黄色和边缘酥脆。
8. 在厨房纸巾上沥油。立即上桌——配大量苹果酱。或甜版：只撒糖粉/肉桂糖。

## 九、总结与关键词

| 关键词 | 释义 |
|------|------|
| **Döner** | 德国最大快餐品类——土耳其起源但德国的"民族化"/本土化 |
| **Currywurst** | 柏林的标志——番茄咖喱酱+咖喱粉+炸香肠切片——90年代德国饮食最"民主"的食物 |
| **Brotzeit** | 面包时间——在巴伐利亚指"冷晚餐"（面包+Wurst+奶酪+啤酒） |
| **Kaffee und Kuchen** | 下午咖啡+蛋糕——德国下午的不可动摇的仪式 |
| **Ostalgie** | 对GDR特产的怀念——Soljanka（汤）、Ketwurst（香肠卷）、特定品牌 |
| **Reinheitsgebot** | 德国啤酒纯净法——德国人对本国啤酒质量骄傲的法律基础 |
| **Imbiss** | 街角小吃亭/站——德国人日常饮食的"第三空间" |

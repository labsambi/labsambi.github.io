# 19世纪意大利常见菜谱汇总

> 本文以**19世纪（1801–1900年）的意大利**为核心——须注意，1861年之前"意大利"并非一个统一国家，而是由撒丁王国、伦巴第-威尼斯（奥地利统治）、教皇国、两西西里王国等多个邦国构成。因此，"意大利菜"在19世纪的绝大部分时间里是**多元地方传统的聚合体**，而非一个统一的菜系。本文参照的核心文献为 **Pellegrino Artusi《厨房科学与饮食艺术》（La scienza in cucina e l'arte di mangiar bene，1891年初版，含475道菜谱；至1910年作者去世前共增补至790道，再版15次）**——该书是意大利统一后第一本真正意义上的"全国菜谱"，被公认为**现代意大利料理的奠基之作**。
>
> 另一个贯穿19世纪意大利饮食的灵魂概念是 **Cucina Povera（穷人厨房）**——以极简的本地食材（面包、橄榄油、豆类、蔬菜、剩菜）做出极致美味的生存智慧，是今天我们认为"正宗"意大利菜的精神内核。

## 一、开胃菜与冷盘（Antipasti）

19世纪意大利的antipasti（字面意为"餐前"）远不如20世纪那样制度化。正式餐厅和富裕家庭的antipasti以腌肉、奶酪和腌菜为主；普通家庭则可能直接跳过头盘，或以一片面包+橄榄油开场。

### 1. 经典Antipasti速览

| 菜名 | 地区 | 描述 |
|------|:--:|------|
| **Prosciutto e Melone（火腿配甜瓜）** | 艾米利亚-罗马涅/伦巴第 | 薄如纸片的生火腿（prosciutto crudo）包裹熟透的甜瓜块，甜与咸的极致对比。Artusi书中记载为夏季经典开胃菜 |
| **Crostini di Fegatini（鸡肝酱烤面包片）** | 托斯卡纳 | 鸡肝+刺山柑+鳀鱼+洋葱用黄油炒熟后研磨成泥状酱（paté），涂在烤脆的小面包片上。托斯卡纳地区节日和日常两用 |
| **Bruschetta（蒜香烤面包片）** | 中部意大利（拉齐奥/翁布里亚） | 厚片乡村面包烤至焦脆，趁热用大蒜瓣摩擦表面，淋上新鲜初榨橄榄油和粗盐。最简单的版本只有这三样——番茄的加入是20世纪初以后的事 |
| **Acciughe al Verde（绿酱鳀鱼）** | 皮埃蒙特 | 盐渍鳀鱼片洗净油浸，浇上欧芹+蒜+橄榄油+醋打成的绿色酱汁，皮埃蒙特的经典antipasto |
| **Olive all'Ascolana（阿斯科利炸橄榄）** | 马尔凯（阿斯科利皮切诺） | 大颗绿橄榄去核，填入肉馅（牛肉+猪肉+帕尔马干酪），裹面包糠油炸。在19世纪已是当地名菜 |
| **Mozzarella in Carrozza（马车夫莫扎瑞拉三明治）** | 坎帕尼亚 | 两片白面包夹莫扎瑞拉奶酪，裹蛋液和面包屑油炸至拉丝。是19世纪那不勒斯利用隔夜面包和剩奶酪的cucina povera杰作 |
| **Insalata di Mare / Frutti di Mare（海鲜沙拉）** | 沿海各地 | 煮熟的虾、蛤蜊、鱿鱼、章鱼用橄榄油、柠檬汁、欧芹凉拌 |
| **Sottaceti e Verdure Sott'Olio（油浸/醋渍蔬菜）** | 各地 | 小洋葱、甜椒、茄子、蘑菇、朝鲜蓟等各类蔬菜切块，用醋或油封存，是全年都有的储藏食品 |

### 2. Antipasto的阶级差异

- **贵族/富人**：摆盘精致的冷盘，可能含鱼子酱（caviale）、鹅肝、多种生火腿、进口法式pâté
- **中产**：本地salami（萨拉米）、一两种奶酪、腌菜（sottaceti）
- **农民**：面包+橄榄油+盐，也许有一片洋葱或腌鳀鱼。Antipasto并非日常概念

## 二、汤类与粥糊（Minestre e Zuppe）

在19世纪意大利农民的餐桌上，汤（minestra / zuppa）往往是全天的唯一热食。Artusi共收录了数十道汤谱，涵盖从奢华的肉汤 tortellini 到一贫如洗的"水煮面包汤"。

### 3. 经典汤品

| 汤名 | 地区 | 描述 |
|------|:--:|------|
| **Tortellini in Brodo（肉汤小馄饨）** | 艾米利亚-罗马涅（博洛尼亚/摩德纳） | 精致小馄饨内填猪肉+火腿+mortadella+帕尔马干酪+肉豆蔻，投入清澈的capon（阉鸡汤高汤中煮熟。节日和周日午餐的第一道菜。Artusi详细描述了捏tortellini的正确技法 |
| **Minestrone（蔬菜浓汤）** | 各地（伦巴第/利古里亚/托斯卡纳版本各不同） | 时令蔬菜（豆类、胡萝卜、洋葱、芹菜、土豆、卷心菜、番茄等）切丁，加米饭或碎意面（pastina）慢煮。热那亚版加罗勒香蒜（pesto）；米兰版加猪皮（cotica）增稠。这是意大利版的"万物锅"——有什么放什么 |
| **Ribollita（重煮汤）** | 托斯卡纳 | 黑羽衣甘蓝（cavolo nero）、白腰豆（cannellini）、土豆、胡萝卜、洋葱熬成的浓汤，关键步骤是加入**隔夜的硬面包**和大量橄榄油再煮一遍（"ribollita" = 重煮）。这是cucina povera的极致：农民将贵族宴席上剩下的面包和肉汤带回家，加入蔬菜"重煮"成自己的晚餐 |
| **Zuppa di Pesce（鱼汤/海鲜汤）** | 沿海各地 | 多种廉价杂鱼（及虾蟹贝类）用橄榄油、大蒜、番茄、白葡萄酒、欧芹熬出的浓汤。利古里亚版本（buridda）、托斯卡纳版本（cacciucco——利沃诺的5鱼汤，每一条鱼代表一个音符）、那不勒斯版本（zuppa di pesce alla napoletana）各擅胜场 |
| **Pasta e Fagioli（意面豆汤）** | 各地 | 豆子（白腰豆/斑豆）+ 短意面（ditalini或碎面条）+ 橄榄油 + 大蒜 + 番茄慢煮。也许是全意大利最普及的家常汤。威尼托版本叫"pasta e fasioi"，托斯卡纳和罗马各有异体 |
| **Passatelli in Brodo** | 艾米利亚-罗马涅 | 面包屑+帕尔马干酪+鸡蛋+柠檬皮+肉豆蔻揉成松软面团，用特制的passatelli压模压入沸腾的高汤中。入口即化，是Romagna地区冬日的安慰食物。Artusi对此有详尽记载 |
| **Zuppa di Cipolle（洋葱汤）** | 皮埃蒙特/伦巴第 | 大量洋葱慢火煎至焦糖化，加入高汤和隔夜面包，表面撒一层磨碎的帕尔马或fontina奶酪 |
| **Acquacotta（熟水）** | 托斯卡纳（马雷马） | 名字极其朴素——"煮过的水"——是牧人和樵夫的野外简餐：洋葱+番茄+水煮成薄汤，打入鸡蛋，将硬面包放在碗底浇上热汤，淋橄榄油。马雷马的牧牛人（butteri）的经典午餐 |

### 4. 汤类阶级差异
- 有肉汤底的（tortellini in brodo, passatelli）是中产以上
- 水+蔬菜+面包（acquacotta, ribollita）是农民和赤贫阶层的日常
- 鱼汤对沿海渔村来说反而是廉价蛋白质来源

## 三、第一道菜——面食（Primi: Pasta）

到了19世纪，意大利面已经从上层阶级的奢侈品变成了全民的主食——这得益于两方面：18世纪末那不勒斯发明了机械压面机（手摇式，配备青铜模具，可量产各种形状）；番茄的大规模种植和普及（18–19世纪）。Artusi的书中，意大利面primi占据了最大篇幅。

### 5. 主要干面食（Pasta Secca）种类

19世纪意大利市面上已经可以购得几十种现成的干面食——**那不勒斯是产业中心**，炎热干燥的海风极其适合晾干面条（格拉尼亚诺Gragnano尤甚）。

| 面型 | 描述 | 经典配法 |
|------|------|------|
| **Spaghetti（意大利面条）** | 长圆面条，此时已是全国最普及的长面型 | 番茄酱、Aglio e Olio（蒜油）、蛤蜊（vongole）、鳀鱼 |
| **Vermicelli（细丝面）** | 比spaghetti更细，意文名=小虫 | 海鲜、番茄酱 |
| **Maccheroni / Maccheroncini（通心粉/小通心粉）** | 19世纪的"maccheroni"泛指短管状面，不一定是今天的弯曲通心粉 | 那不勒斯番茄肉酱（ragù napoletano）、焗烤 |
| **Penne（笔管面）** | 斜切管状面，得名于鹅毛笔尖（penna） | 番茄酱、arrabbiata辣酱 |
| **Rigatoni（条纹粗管面）** | 粗管面，表面有纵向条纹，19世纪随机器制面而出现 | 番茄肉酱、焗菜（pasta al forno） |
| **Fettuccine（缎带面）** | 手工鸡蛋宽面（约1cm宽），罗马和拉齐奥地区的标志 | 黄油+帕尔马干酪（fettuccine al burro）、ragù romano |
| **Tagliatelle（塔利亚泰莱面）** | 博洛尼亚的鸡蛋宽面（约8mm宽），博洛尼亚烹饪协会规定煮后宽度为8mm（等于La Grassa塔楼的1/12270高度） | **Ragù Bolognese**（博洛尼亚肉酱——但19世纪的版本和今天的完全不同，见下文酱料章） |
| **Lasagne（千层面）** | 长方形面片，多层叠放焗烤 | 博洛尼亚式（菠菜面片+ragù+besciamella白酱+帕尔马干酪）；那不勒斯式（ricotta+肉丸+莫扎瑞拉+煮蛋片） |
| **Pappardelle（宽缎带面）** | 极宽的手工鸡蛋面（约2–3cm），托斯卡纳特产 | 野猪肉酱（cinghiale）、野兔肉酱（lepre） |
| **Trenette（利古里亚扁长面）** | 利古里亚地区的扁长面，类似linguine | **Pesto alla Genovese**（热那亚罗勒香蒜酱）+ 土豆+ 青豆角 |
| **Ditalini / Pastina（小粒面）** | 管状或星形的小粒面，用于汤中 | 豆汤（Pasta e Fagioli）、清汤 |
| **Orecchiette（小耳朵面）** | 普利亚的标志。用拇指在案板上按压出的凹形小饼，因形状似耳朵而得名 | 西兰花（cime di rapa）+ 鳀鱼 + 大蒜 + 辣椒碎 |

### 6. 手工鲜面食（Pasta Fresca）与填馅面

手工鸡蛋面以软质小麦粉（farina di grano tenero）+鸡蛋为主要原料，是意大利北部和中部的传统。南部则更多用硬质小麦粉（semola）+水，且制成干面食保存。

| 面型 | 地区 | 描述 |
|------|:--:|------|
| **Ravioli（方饺）** | 各地（皮埃蒙特、托斯卡纳、艾米利亚皆有） | 两片面皮夹馅（最经典的是ricotta奶酪+菠菜，称为"ravioli di magro"即斋版饺），用齿轮刀切开封边。也有的填肉馅。19世纪ravioli的形状和大小各地差异极大 |
| **Tortellini（小馄饨/肚脐饺）** | 艾米利亚-罗马涅 | 小方片面皮包入肉馅（猪肉+火腿+mortadella+帕尔马干酪+鸡蛋+肉豆蔻），折叠成三角形后两角相扣，形成独特的肚脐/花苞形状。按传说，其形状来自对维纳斯肚脐的模仿。Artusi的版本非常详细 |
| **Agnolotti / Agnellotti（皮埃蒙特方饺）** | 皮埃蒙特 | 小方饺内填烤牛肉/猪肉+菠菜+帕尔马干酪+鸡蛋。通常以烤肉时滴下的汁（sugo d'arrosto）或黄油+鼠尾草调味 |
| **Cappelletti（小帽子饺）** | 罗马涅/马尔凯 | 比tortellini略大，形状像小帽子。馅料含ricotta和帕尔马干酪（斋版）或肉馅（荤版），圣诞节期间浮在capon肉汤中食用 |
| **Cannelloni（粗管面）** | 坎帕尼亚/各地 | 长方形势蛋面片，煮熟后卷入填馅（肉碎+ricotta+菠菜等），排列在烤盘中，淋白酱（besciamella）和番茄酱，撒帕尔马干酪焗烤。19世纪下半叶流行于那不勒斯和罗马 |

### 7. 经典面食做法举例

#### Tagliatelle al Ragù Bolognese（博洛尼亚肉酱宽面）—— 19世纪版本

**注意**：19世纪的ragù bolognese和今天遍布全球的"肉酱意大利面"（spaghetti bolognese——在意大利根本不存在这道菜）**完全不同**。

- **面**：手工鸡蛋宽面（tagliatelle），绝对不能是spaghetti。
- **19世纪肉酱核心**（按Artusi及更早的博洛尼亚配方）：
  - 极少量番茄浓缩膏（conserva di pomodoro）甚至无番茄
  - 核心是细切的牛肉（不是肉末！）+ pancetta（意式培根）丁 + 洋葱丁 + 胡萝卜丁 + 芹菜丁（soffritto基础）
  - 用肉高汤（brodo di carne）慢炖数小时，末尾加入少量牛奶或奶油
  - 上桌前撒大量帕尔马-雷焦干酪（Parmigiano-Reggiano）
- **与今天的区别**：今天的ragù番茄更多、肉是碎肉末而非小肉块、常用红酒（19世纪不一定加）。

#### Spaghetti al Pomodoro（番茄意大利面）—— 19世纪经典版

Artusi收录了"Maccheroni alla Napoletana"。番茄在18世纪中后期开始在意大利南部广泛种植食用，到了19世纪已经是南部（尤其是那不勒斯）面食的核心酱料。

- 熟透的番茄（pomodori San Marzano最佳）用橄榄油+大蒜+盐慢火熬成浓厚的汁（sugo），出锅加新鲜罗勒叶。极简主义的巅峰，一道纯素食的完美面食。

#### Spaghetti Aglio, Olio e Peperoncino（蒜油辣面）

- 大蒜切片在大量橄榄油中小火煸至微金，加入干辣椒碎（peperoncino），淋在刚煮好的spaghetti上，撒欧芹碎。那不勒斯贫民区的深夜食物——便宜、快速、饱腹。

### 8. 焗烤面食（Pasta al Forno）

| 菜名 | 描述 |
|------|------|
| **Lasagne alla Bolognese（博洛尼亚千层面）** | 绿色菠菜面片 + ragù肉酱 + 白酱（besciamella）+ 帕尔马干酪，层层叠放焗烤 |
| **Pasta al Forno alla Napoletana（那不勒斯焗面）** | 短管面（如ziti）+ 番茄酱 + 小肉丸（polpettine）+ ricotta奶酪 + 莫扎瑞拉奶酪 + 煮蛋片 + salame，节日或周日午餐 |
| **Timballo di Maccheroni（通心粉大烤饼）** | 贵族/中产宴会的炫耀性大菜——短意面用甜面包壳或千层酥皮（pasta sfoglia）包裹，中间夹肉馅、蘑菇、奶酪、煮蛋、肉丸，烤成圆塔形。西西里Palermo的电影《豹》（Il Gattopardo）中的timballo便是这种菜的19世纪贵族版缩影 |

## 四、第一道菜——米饭、玉米粥与面疙瘩（Risotto, Polenta, Gnocchi）

意大利北部（尤其以波河平原为核心）的气候和灌溉渠系统适合种水稻和玉米，因此米和玉米粥在伦巴第、皮埃蒙特、威尼托的作用相当于南部的面食。

### 9. Risotto（意大利烩饭）

Risotto（烩饭）作为独立烹饪技艺，在19世纪中期基本定型。其核心是一边搅拌一边少量多次地加入热高汤，让米粒在吸收液体的同时释放淀粉，最终形成creamy的口感——**不用任何奶油**（这是现代新发明的"捷径"），全靠米种和搅动的功夫。

| 烩饭名 | 地区 | 描述 |
|------|:--:|------|
| **Risotto alla Milanese（米兰藏红花烩饭）** | 米兰/伦巴第 | **意大利最著名的risotto**。据传16世纪为米兰大教堂的玻璃彩绘师发明的玩笑——在婚宴的risotto中加了藏红花（saffron）——但成菜金黄华丽，从此成为经典。19世纪版本：牛骨髓（midollo）+黄油炒洋葱，加入arborio/carnaroli米，边搅边加牛肉高汤，最后混入藏红花丝和大量帕尔马干酪。常与**Ossobuco**（炖小牛膝，见下文肉类章）一起上桌 |
| **Risotto alla Piemontese（皮埃蒙特烩饭）** | 皮埃蒙特 | 用本地米在黄油中炒至微透，加牛肉高汤慢煮，上桌前加大量黄油和帕尔马干酪快速搅拌（mantecatura）。有时加白松露薄片——阿尔巴（Alba）的白松露在19世纪已为美食家珍视 |
| **Risi e Bisi（米和豌豆）** | 威尼托 | 介于汤和risotto之间——用豌豆（bisi，威尼斯方言）+ pancetta + 洋葱 + 欧芹煮成湿润的米饭菜肴。威尼斯总督府的春季传统菜肴，也是19世纪威尼斯家常菜 |
| **Risotto alla Certosina（卡尔特会修会烩饭）** | 伦巴第 | 斋戒版risotto——不用肉高汤，用水或蔬菜汤代替。含淡水小龙虾（gamberi di fiume）、青蛙腿、豌豆和蘑菇。Artusi记载为修道院菜谱 |

**Artusi的risotto秘诀**（1891年）：
> 米必须在黄油中"烤"（tostare）至微透但不焦，搅动要用木勺，高汤一勺一勺地加，不能急。成品应该"all'onda"（像波浪一样在盘中微微流动），而不是硬邦邦的一团。

### 10. Polenta（玉米粥）

在玉米被引入欧洲（16–17世纪）后，玉米粥（polenta）逐渐取代了中世纪的杂粮粥（pulmentum，以大麦、斯佩耳特小麦、栗子粉等制成）。到了19世纪，polenta已经是伦巴第、威尼托、弗留利、特伦蒂诺等北部地区的**绝对主食**，其重要性不亚于南部的面包——但对于赤贫阶层来说，几乎顿顿只吃玉米粥也导致了糙皮病（pellagra，烟酸缺乏症）的流行（尤其是在威尼托和伦巴第农村）。

| 类型 | 描述 |
|------|------|
| **Polenta Gialla（黄玉米粥）** | 黄玉米粉（farina di mais）撒入沸腾的盐水，用大铜锅（paiolo）以长木棍不停搅拌50–60分钟，直到木棍能立在粥中不倒。倒入木板（tavola）摊平，冷却后切块 |
| **Polenta Taragna（荞麦玉米粥）** | 伦巴第/瓦尔泰利纳（Valtellina）：玉米粉混合荞麦粉（grano saraceno），煮成深色粥。配大量黄油和本地奶酪（如Bitto或Casera）融化其中 |
| **Polenta Concia（浇汁玉米粥）** | 瓦莱达奥斯塔（Valle d'Aosta）：层叠的玉米粥片+ fontina奶酪+黄油，放入烤箱烤至奶酪起泡 |
| **Polenta e Osei（玉米粥配小鸟）** | 伦巴第：烤小禽（云雀、麻雀等）放在玉米粥上，浇上黄油和鼠尾草汁。今已禁止捕鸟，由巧克力杏仁糕做成的"假鸟"替代 |
| **Polenta Fritta（炸玉米粥条）** | 剩玉米粥切条入油锅炸至外脆内软，配salami或奶酪，是北部农民冬季的简单午餐 |

**19世纪悲惨的一面：** 玉米粥的单调饮食（几乎无蛋白质和维生素来源）导致糙皮病在威尼托、伦巴第的教区记录中被大量记载——1860–1880年代，意大利政府开始推广多样化饮食以对抗这一公共健康危机。

### 11. Gnocchi（面疙瘩/土豆团子）

| 菜名 | 地区 | 描述 |
|------|:--:|------|
| **Gnocchi di Patate（土豆团子）** | 各地（北部尤多） | 土豆在意大利的广泛种植是18–19世纪的事，因此土豆gnocchi也是相对"新"的食物。做法：煮熟的土豆压泥+面粉+鸡蛋+少许盐，揉成软面团，搓长条切小段，用叉背压出凹纹。沸水煮至浮起。配黄油+鼠尾草、番茄酱、或pesto |
| **Gnocchi alla Romana（罗马式粗面疙瘩）** | 罗马/拉齐奥 | 这与上面的土豆团子**完全不同**——用粗粒麦粉（semolino）煮成厚糊状，摊平冷却后切成圆形片，一层层排列在烤盘中，每层撒帕尔马干酪和黄油，入烤箱烤至表面金黄。Artusi对此有经典描述。这实际上是古代罗马的puls（麦糊粥）的直系后代 |
| **Gnocchi di Zucca（南瓜团子）** | 曼托瓦（Mantova）/伦巴第 | 南瓜泥替代部分土豆，成品更甜更软，配融化黄油+鼠尾草+帕尔马干酪 |

## 五、披萨与街头面食（Pizza e Cibo di Strada）

### 12. Pizza Napoletana（那不勒斯披萨）

19世纪的那不勒斯是一座拥挤、炎热、贫穷的城市，披萨正是为这样的城市而生的食物——便宜、快捷、可以在街头站着吃。

**历史背景：** 披萨的前身早在古代就已存在（各种flatbread），但现代意义上的披萨（面团+番茄+奶酪）在那不勒斯的定型是在18–19世纪。最早的文字记载中提到披萨作为"穷人的食物"，以大蒜+猪油+盐（pizza all'aglio e oglio）或番茄+罗勒+少许奶酪为调料。

**1889年传奇：** 据传，意大利国王翁贝托一世和王后玛格丽特（Margherita di Savoia）到访那不勒斯，当地披萨师傅 **Raffaele Esposito** 为致敬新统一的意大利，创作了一款以国旗三色为灵感的披萨——红（番茄）、白（莫扎瑞拉）、绿（罗勒）——并将其命名为 **Pizza Margherita**。这一故事的真伪有争议（有些历史学家认为Margherita pizza在那之前就已经存在），但无论如何，**Pizza Margherita 由此成为意大利统一的国家象征**。

| 披萨种类（19世纪末） | 描述 |
|------|------|
| **Pizza Margherita** | 面饼抹San Marzano番茄酱（passata），铺上水牛莫扎瑞拉（mozzarella di bufala）薄片或碎块，撒盐和橄榄油，在木火砖窑中烤90秒。出炉后加新鲜罗勒叶。极简，极美 |
| **Pizza Marinara（水手披萨）** | **更古老的款式**，无奶酪（纯素/斋版）。番茄酱+大蒜+牛至叶（origano）+橄榄油。传说水手出海前吃这个，因为保存期长 |
| **Pizza Napoli / Romana** | 在罗马等地用鳀鱼（acciuga）+刺山柑（capperi）+橄榄的披萨，到了19世纪末逐渐形成地方风格 |
| **Pizza Bianca（白披萨）** | 无番茄——仅以橄榄油+粗盐+迷迭香为调料的面饼。罗马的pizza bianca是elongated的厚软面饼，不同于那不勒斯圆形薄底 |

**披萨的社会地位：** 在19世纪，披萨几乎仅在那不勒斯的工人阶级和街头饮食习惯中存在。意大利其他地方的人很少知道披萨为何物——它是纯粹的那不勒斯本地食物。直到二战后，披萨才开始传遍意大利全国和全球。Artusi（佛罗伦萨人）的1891年初版中甚至没有提到披萨——他后来加了一个"pizza alla napoletana"的食谱，但那实际上是一种甜点蛋糕（ricotta馅的烤饼），和今天的披萨不相干。

### 13. 其他街头食品

| 食品 | 地区 | 描述 |
|------|:--:|------|
| **Frittatine di Maccheroni（通心粉煎饼）** | 那不勒斯 | 剩下的煮通心粉（或spaghetti）混入鸡蛋和奶酪，做成小煎饼，炸或煎至金黄。完美利用剩面的街头小吃 |
| **Panzerotti（油炸馅饼）** | 普利亚 | 半月形油炸面饼，内填番茄酱和mozzarella——本质上是迷你封口油炸披萨 |
| **Farinata / Cecina（鹰嘴豆薄饼）** | 利古里亚（热那亚）/ 托斯卡纳 | 鹰嘴豆粉+水+橄榄油+盐调成稀面糊，在极热的大铜盘中烤成薄脆饼。无麸质，口感类似介于pancake和脆饼之间。在热那亚叫farinata，在托斯卡纳叫cecina |
| **Panelle（鹰嘴豆煎饼）** | 西西里（巴勒莫） | 鹰嘴豆粉+水+欧芹煮成厚糊，摊平冷却切条，油炸至金黄。巴勒莫的经典街头小吃，常夹入芝麻面包中（pane e panelle） |
| **Scagliozzi（油炸玉米粥块）** | 普利亚 | 剩的硬玉米粥切块，油炸至外酥脆 |
| **Stigghiola（烤羊肠）** | 西西里（巴勒莫） | 羊肠洗净后缠绕在韭葱茎或欧芹梗上，炭火烤至焦脆。气味强烈但粉丝众多 |
| **Lampredotto（牛第四个胃三明治）** | 佛罗伦萨 | 牛的第四个胃（皱胃，lampredotto）在番茄+洋葱+芹菜高汤中慢煮至软嫩，切薄片夹入浸过肉汤的面包卷（semelle）中，配绿酱（salsa verde，欧芹+刺山柑+鳀鱼+醋）或辣酱 |

## 六、酱料（Sughi e Ragù）

### 14. 核心酱料一览

| 酱 | 地区 | 关键原料 | 19世纪用法 |
|------|:--:|------|------|
| **Ragù Bolognese（博洛尼亚肉酱）** | 艾米利亚-罗马涅 | 牛肉（细切非肉末）、pancetta（意式培根）、soffritto（洋葱+胡萝卜+芹菜）、少量番茄浓缩膏、高汤、牛奶、红酒 | 配**手工鸡蛋宽面**（tagliatelle）或绿色lasagne。19世纪配方中番茄含量远低于今天。**绝不**配spaghetti！ |
| **Ragù Napoletano（那不勒斯肉酱）** | 坎帕尼亚 | 大块牛肉（牛胸或牛肩）或者猪肋骨、salsiccia香肠、洋葱、红酒、大量番茄酱passata | 慢炖4–6小时。肉块捞出作为"secondo"（第二道菜）单独上桌，留下的酱汁用来拌面（primo）。一锅两吃——cucina povera的精明 |
| **Pesto alla Genovese（热那亚罗勒香蒜酱）** | 利古里亚 | **仅6种原料**：新鲜罗勒叶、松子（pinoli）、大蒜、帕尔马干酪（或撒丁pecorino）、粗海盐、Taggia品种的特级初榨橄榄油。用大理石研钵和木棒**手工研磨**（绝对不能blender！） | 配trenette或trofie面食，传统上同锅煮土豆丁和青豆角 |
| **Sugo di Pomodoro / Salsa di Pomodoro（番茄酱）** | 南部（坎帕尼亚为首） | San Marzano长番茄、橄榄油、大蒜、盐、罗勒 | 19世纪最基础的意大利面酱。番茄是16世纪从美洲传入的，但直到18–19世纪才在南意大利大规模种植食用 |
| **Sugo all'Arrabbiata（愤怒酱）** | 拉齐奥（罗马） | 番茄酱底+大量干辣椒碎+大蒜 | 意为"愤怒的酱料"，来自其刺激的辣度 |
| **Sugo alla Puttanesca（妓女酱）** | 坎帕尼亚/拉齐奥 | 番茄酱+鳀鱼+黑橄榄（Gaeta品种）+刺山柑+大蒜+辣椒碎 | 名字来源众说纷纭，一种说法是此酱用料都是储藏柜中的"干货"，连妓女在一轮客人之间也能快速做好 |
| **Salsa Verde（绿酱）** | 各地（皮埃蒙特/伦巴第/艾米利亚） | 欧芹+刺山柑+鳀鱼+大蒜+蛋黄/面包+橄榄油+醋 | 配煮肉（bollito misto）、牛舌、鱼，是北意大利万能冷酱 |
| **Salsa di Noci（核桃酱）** | 利古里亚 | 核桃仁+松子+面包浸牛奶+大蒜+橄榄油+Parmigiano | 利古里亚另一种经典面酱，配pansotti（三角形填馅面）或trofie |
| **Burro e Salvia（黄油鼠尾草酱）** | 北部各地 | 黄油+新鲜鼠尾草叶 | 最简单的调味：黄油在小锅中加热至微焦，放入鼠尾草叶子炸脆，浇在ravioli、gnocchi或agnolotti上 |
| **Besciamella（白酱/法式奶酱）** | 受法餐影响，北意大利通用 | 黄油+面粉+牛奶+肉豆蔻 | Lasagne焗烤层、cannelloni、crespelle |

## 七、第二道菜——肉类（Secondi di Carne）

19世纪意大利的肉类消费高度依赖社会经济地位：农民一年可能只吃几次肉（节日）；城市中产每周数次；贵族和富人几乎顿顿有肉——但以野味（selvaggina）和小牛肉（vitello）为上品，猪肉和廉价部位留给下层。

### 15. 北部肉类菜肴

| 菜名 | 地区 | 描述 |
|------|:--:|------|
| **Ossobuco alla Milanese（米兰炖小牛膝）** | 米兰/伦巴第 | 横切的小牛膝骨（中心的骨髓是精华所在），裹面粉黄油煎至金黄，与soffritto（洋葱+胡萝卜+芹菜）、白葡萄酒、少量番茄和高汤慢炖至肉酥烂，最后撒gremolata（柠檬皮碎+欧芹+大蒜的干性调味）。传统上配Risotto alla Milanese，牛膝骨的骨髓挖出来拌进金黄色米饭中——一口天堂 |
| **Cotoletta alla Milanese（米兰炸小牛排）** | 米兰/伦巴第 | 带骨的小牛肋排（不拍平）+鸡蛋液+面包屑（不加帕尔马干酪——米兰传统坚持此原则），用澄清黄油煎至外酥内粉红。有人说是奥地利Wiener Schnitzel的变体（伦巴第曾被奥地利统治），但米兰人坚称其更早。做成极薄的版本叫"orecchia d'elefante"（大象耳朵） |
| **Bollito Misto（混合煮肉）** | 皮埃蒙特 | 多种肉类（牛肉、小牛舌、牛头肉、capon鸡、cotechino香肠）在香料蔬菜汤中慢煮数小时。上桌时切片，配多种酱料：salsa verde（绿酱）、mostarda di Cremona（糖渍芥末水果）、bagnetto rosso（红椒酱）、辣根酱。皮埃蒙特的宴席硬菜 |
| **Brasato al Barolo（Barolo红酒炖牛肉）** | 皮埃蒙特 | 大块牛肉用皮埃蒙特最负盛名的Barolo红酒+蔬菜+香料低温慢炖8小时以上，肉质酥烂如butter。这是最奢侈的皮埃蒙特冬季菜——好酒入菜 |
| **Vitello Tonnato（金枪鱼酱小牛肉）** | 皮埃蒙特 | 煮小牛肉冷切薄片，淋上金枪鱼+鳀鱼+刺山柑+蛋黄酱+柠檬汁打成泥的奶油状酱。夏季经典冷荤菜 |
| **Cassoeula（炖卷心菜猪肉）** | 伦巴第 | 猪的各个部位（蹄、皮、肋排、香肠——尽用边角）与大叶子卷心菜（verza）在陶锅中慢炖。米兰冬季的灵魂家常菜，传说有猪的地方就有cassoeula，从圣安东尼节后开始吃 |
| **Fegato alla Veneziana（威尼斯式炒肝）** | 威尼托 | 小牛肝薄片与大量洋葱在黄油+橄榄油中快速翻炒，加白葡萄酒收汁。洋葱必须彻底炒软至焦糖化，为肝脏的"金属味"提供温柔甜美的陪衬 |

### 16. 中部与南部肉类菜肴

| 菜名 | 地区 | 描述 |
|------|:--:|------|
| **Bistecca alla Fiorentina（佛罗伦萨大牛排）** | 托斯卡纳 | 奇亚尼纳（Chianina）品种白牛的T骨牛排，厚约4指宽，在橡木炭火上短暂炙烤——外层焦香、内里近乎生（"al sangue"）。仅以盐、橄榄油、也许一瓣蒜擦拭，拒绝任何花哨酱料。是一道考验牛肉品质和师傅火候的极简菜。Artusi记载这道菜是"英国人教给佛罗伦萨人的"——因为英国人在19世纪大量旅行托斯卡纳 |
| **Saltimbocca alla Romana（罗马跳嘴小牛肉卷）** | 罗马/拉齐奥 | 薄小牛肉片上铺火腿薄片（prosciutto crudo）和新鲜鼠尾草叶，卷起用牙签固定，黄油煎至金黄，白葡萄酒+少量水收汁。"Saltimbocca"意为"跳进嘴里"——因为太好吃了 |
| **Abbacchio alla Romana（罗马烤乳羊）** | 罗马/拉齐奥 | 乳羊肉（abbacchio）切块，用迷迭香、大蒜、橄榄油、鳀鱼酱腌制，烤至皮酥肉嫩，配烤土豆。罗马复活节传统菜 |
| **Trippa alla Romana（罗马式炖牛肚）** | 罗马/拉齐奥 | 牛肚（trippa）切条，与番茄酱、薄荷（mentuccia romana，本地野生薄荷品种）、pecorino romano奶酪慢炖。罗马工人阶级的周六经典午餐 |
| **Polpette / Polpettone（肉丸/大肉饼）** | 各地 | 肉末+面包屑浸牛奶+鸡蛋+帕尔马干酪+欧芹，做成小丸子炸/煎，或做成大肉饼（polpettone，内可藏入煮蛋），配番茄酱 |
| **Coda alla Vaccinara（屠夫式炖牛尾）** | 罗马 | 牛尾切段与番茄、芹菜（大量）、红酒、松子、葡萄干慢炖至胶质融化，肉从骨上分离。源自罗马屠宰场（Mattatoio di Testaccio）工人的quinto quarto（第五部分 = 内脏边角）文化 |
| **Porchetta（烤填馅乳猪/大猪）** | 中部意大利（拉齐奥/翁布里亚/阿布鲁佐） | 整头去骨猪（数十斤重），内腔填入野茴香花（finocchietto selvatico）、迷迭香、大蒜、盐、黑胡椒、猪肝后缝合，在旋转烤架上烤至皮脆得如玻璃般透明。集市和节庆时切片售卖，夹入面包便是完美街头午餐 |

### 17. 野味（Selvaggina）

| 菜名 | 描述 |
|------|------|
| **Cinghiale in Umido（炖野猪）** | 托斯卡纳/翁布里亚：野猪肉用红酒+番茄+杜松子+月桂叶长时间慢炖，去除腥膻。配pappardelle宽面或polenta |
| **Lepre in Salmì（红酒炖野兔）** | 皮埃蒙特/伦巴第：野兔先在红酒+蔬菜+香料中腌制24–48小时，再连同腌料慢炖，最后以磨碎的兔肝增稠酱汁 |
| **Fagiano / Pernice / Quaglia（野鸡/鹧鸪/鹌鹑）** | 贵族餐桌上的野禽——烤制、填入馅料、或做成肉冻（gelatina） |
| **Tordi / Allodole（画眉/云雀）** | 小野禽——用猪网油（rete di maiale）包裹，炭火烤制，配polenta |

## 八、第二道菜——鱼类与海鲜（Secondi di Pesce）

意大利有超过7,600公里的海岸线，鱼类和海鲜在南意大利、西西里、撒丁岛、利古里亚和威尼托的饮食中地位尤其重。斋戒期（周五、大斋期等）鱼类需求旺盛。

### 18. 经典海鲜菜肴

| 菜名 | 地区 | 描述 |
|------|:--:|------|
| **Baccalà alla Vicentina（维琴察式炖鳕鱼干）** | 威尼托（维琴察） | 挪威进口的鳕鱼干（stoccafisso，风干非盐腌），在水中浸泡数日复原，与洋葱、鳀鱼、牛奶、橄榄油在陶罐中文火慢炖4–5小时。配奶油状polenta。这道菜历史悠久，16世纪特伦托大公会议的菜单上就有记载。19世纪维琴察地区已将其视为城市骄傲 |
| **Baccalà alla Livornese（利沃诺式炖咸鳕鱼）** | 托斯卡纳（利沃诺） | 咸鳕鱼（baccalà，盐腌版）浸水脱盐后，与番茄、黑橄榄、刺山柑慢炖 |
| **Fritto Misto di Pesce（混合炸海鲜）** | 坎帕尼亚/利古里亚/各地沿海 | 小小的整鱼（triglia红鲻鱼、merluzzetti小鳕鱼等）、鱿鱼圈、小虾、沙丁鱼——全裹上薄粉，快速油炸至脆。撒盐和柠檬汁。海边城镇的日常 |
| **Sarde in Saor（酸甜腌沙丁鱼）** | 威尼托 | 沙丁鱼炸至酥脆，层层铺入盘中，每层夹大量的洋葱（慢炒至焦糖化）+ 松子 + 葡萄干 + 白葡萄酒醋。封好后在地窖中静置数日，酸甜交融。在威尼斯，游船夫（gondoliere）的传统携带食物 |
| **Cacciucco alla Livornese（利沃诺海鲜浓汤）** | 托斯卡纳（利沃诺） | "五鱼汤"——用五种以上的鱼类和海鲜（最廉价的小杂鱼+墨鱼+虾+贝类），番茄、大蒜、辣椒碎、红酒熬成浓稠红汤，倒在烤面包片上。传统上要用恰好5种鱼，代表利沃诺五音阶 |
| **Insalata di Polpo（章鱼沙拉）** | 各地沿海 | 章鱼在盐水中煮至柔软，切块，用橄榄油+柠檬汁+欧芹+芹菜丁凉拌。热那亚、那不勒斯、西西里各有版本 |
| **Cozze alla Marinara（水手式焗青口）** | 坎帕尼亚/普利亚 | 新鲜青口贝（cozze / mitili）在锅中与大蒜、橄榄油、白葡萄酒、欧芹、番茄丁同煮至壳开，配烤面包片蘸汤汁 |
| **Seppie col Nero（墨汁墨鱼）** | 威尼托/弗留利 | 墨鱼（seppie）带墨囊一起与番茄酱慢炖，黑色的酱汁用来拌polenta或意面。威尼斯潟湖的经典 |
| **Sgombro / Triglie / Orate / Spigola（青花鱼/红鲻鱼/金头鲷/海鲈）** | 各地 | 整鱼撒盐和迷迭香，橄榄油淋面，烤架或烤炉烹制。鱼类新鲜度决定一切——"pesce"（鱼）的灵魂在于"fresco"（新鲜） |

### 19. 西西里海岛鱼

西西里环抱地中海，其海鲜传统独特而丰富：

| 菜名 | 描述 |
|------|------|
| **Pesce Spada alla Siciliana（西西里剑鱼）** | 剑鱼排用橄榄油+柠檬+牛至+刺山柑腌制后烤/煎。墨西拿海峡是地中海剑鱼的传统渔场 |
| **Sarde a Beccafico（沙丁鱼卷）** | 巴勒莫名菜：新鲜沙丁鱼片去骨展开，上面铺面包屑+松子+葡萄干+鳀鱼+橙皮+欧芹的甜咸馅，卷起固定，排列在盘中淋橄榄油和橙汁，烤至金黄 |
| **Polpo alla Pignata（陶罐炖章鱼）** | 章鱼在陶罐（pignata）中以自己的汁液慢炖，无需加水——章鱼释放的水分足以形成浓郁的汤汁 |

## 九、蔬菜与配菜（Contorni e Verdure）

### 20. 经典蔬菜配菜

| 菜名 | 地区 | 描述 |
|------|:--:|------|
| **Caponata（西西里酸甜炖菜）** | 西西里 | 茄子（主料）+ 芹菜 + 洋葱 + 橄榄 + 刺山柑 + 番茄用橄榄油炸或炖煮，最后加入醋和少量糖——关键在"agrodolce"（酸甜）的平衡。热吃冷吃均可 |
| **Peperonata（甜椒炖菜）** | 拉齐奥/南部 | 红黄甜椒切条，与洋葱、番茄在橄榄油中慢炖至软滑 |
| **Carciofi alla Romana（罗马式炖朝鲜蓟）** | 拉齐奥 | 罗马朝鲜蓟（carciofi romaneschi，球形无刺）挖去内芯，填塞mentuccia（罗马野生薄荷）+大蒜+欧芹碎，整颗在橄榄油+水中焖煮至软 |
| **Carciofi alla Giudia（犹太式炸朝鲜蓟）** | 拉齐奥 | 罗马犹太人区（Ghetto di Roma）的创造——整颗朝鲜蓟按压后两次油炸至花瓣焦脆如薯片，撒盐和黑胡椒。16世纪已存在，19世纪成为罗马的跨族群经典 |
| **Melanzane alla Parmigiana（帕尔马式焗茄子）** | 坎帕尼亚/西西里（非帕尔马！） | 名字中的"Parmigiana"指帕尔马干酪，而非帕尔马城。茄子切片油炸，层层堆叠：茄子 → 番茄酱 → mozzarella → 帕尔马干酪 → 罗勒 → 再一层茄子，入烤箱焗烤。Artusi记载为西西里菜 |
| **Fiori di Zucca Fritti（炸南瓜花）** | 罗马/各地 | 嫩南瓜花（fiori di zucca）裹薄面糊（pastella）快速油炸。有时先填入一小块mozzarella和鳀鱼 |
| **Insalata di Pomodori（番茄沙拉）** | 南部 | 熟番茄切片+罗勒叶+橄榄油+盐——最简单的夏季配菜 |
| **Verdure Grigliate（烤蔬菜）** | 各地 | 茄子、甜椒、西葫芦切片，在炭火烤架上烤到条纹焦痕，淋橄榄油和醋 |

### 21. Cicoria / Catalogna（苦菊/蒲公英叶）

苦味蔬菜在意大利中部和南部的饮食中扮演了重要角色——不是 salad，而是作为热配菜（contorno）：

- **Cicoria Ripassata in Padella（回锅苦菊）**：野生菊苣（cicoria selvatica）煮至半软，用大量大蒜+辣椒碎+橄榄油在锅中猛火"回炒"（ripassare）。罗马工人阶级餐桌上的日常。
- **Puntarelle alla Romana**：Catalogna菊苣的嫩芽切成细丝（在冰水中会卷曲成螺旋形），拌上橄榄油+大蒜+鳀鱼+醋做成的vinaigrette。

## 十、腌肉（Salumi）

意大利各地都有本地的猪肉屠宰传统（"del maiale non si butta via niente"——猪身上没有可丢弃的东西），salumi是这种智慧的集大成产物。19世纪，每个村庄都有自己引以为豪的salami、火腿和猪油。

### 22. 主要Salumi种类

| 名称 | 地区 | 描述 |
|------|:--:|------|
| **Prosciutto Crudo（生火腿）** | 帕尔马（Parma）、圣丹尼尔（San Daniele / 弗留利）、托斯卡纳 | 整条猪后腿用海盐腌渍后长时间风干熟成（12–36个月）。帕尔马火腿（Prosciutto di Parma）在19世纪已闻名全欧，以其甜味和入口即化的口感著称。切成极薄片，近乎透明 |
| **Prosciutto Cotto（熟火腿）** | 各地 | 煮熟的调味猪腿，比crudo更接近今天的ham |
| **Mortadella Bologna（博洛尼亚大香肠）** | 博洛尼亚/艾米利亚-罗马涅 | 猪肉+猪肥肉丁（lardelli，切成立方体）+黑胡椒粒+没药（mirto）做成的大直径圆柱香肠，慢火蒸煮。切面有白色脂肪斑点均匀散布。19世纪博洛尼亚已为此制定了严格的官方标准 |
| **Salame（萨拉米）** | 各地（品种逾百） | 猪肉+肥肉+盐+香料（黑胡椒、大蒜、红酒等）灌入天然肠衣，在特定温湿度中风干发酵数周到数月。每个地区甚至每个家庭都有自己的配方。19世纪是各地salami百花齐放的时期 |
| **Finocchiona** | 托斯卡纳 | 加入大量茴香籽（finocchio）的粗粒salami，据传中世纪的托斯卡纳酒商用茴香籽替代昂贵的黑胡椒 |
| **Pancetta（意式培根）** | 各地 | 猪五花肉用盐和香料腌渍后卷成圆柱形风干，是carbonara、amatriciana等经典酱料的灵魂基材 |
| **Guanciale（猪脸颊肉腌肉）** | 拉齐奥 | 猪脸颊（guancia）腌制风干，肥瘦比例极高，熔化后释出浓郁的猪肉风味。这是正宗的Carbonara和Amatriciana的"唯一指定用肉"——不用pancetta更不用bacon |
| **Lardo di Colonnata（科隆纳塔白猪油）** | 托斯卡纳（卡拉拉大理石矿区） | 猪背部肥肉整块埋入卡拉拉白色大理石槽中，层叠海盐+迷迭香+鼠尾草+大蒜+多种香辛料，在大理石的恒温恒湿环境中熟成6–12个月。成品白如大理石，入口如丝绸般融化。在19世纪是矿工的高能量口粮 |
| **Coppa / Capocollo** | 各地（皮亚琴察、卡拉布里亚等） | 猪颈肉经盐渍+香料+风干，肉质软嫩，大理石油花丰富 |
| **Speck Alto Adige** | 南蒂罗尔 | 结合了北欧烟熏传统和意大利风干技艺——猪腿先冷烟熏再风干。奥匈帝国时期传入 |
| **Cotechino / Zampone（猪皮香肠/填充猪脚）** | 摩德纳/艾米利亚 | 猪肉+猪皮+香料灌入肠衣（cotechino）或填回猪脚皮中（zampone）。冬季食用，尤其是跨年夜配小扁豆（lenticchie）——豆子象征硬币，带来财运 |
| **Nduja** | 卡拉布里亚 | 极辛辣的可涂抹发酵猪肉酱（含大量的卡拉布里亚辣椒），灌入肠衣风干。用于拌面或涂面包 |
| **Salsiccia（鲜香肠）** | 各地 | 粗粒猪肉+盐+黑胡椒+茴香籽灌入肠衣，可烤可煎可入ragù中同煮。每个地区有自己的香肠配方 |
| **Sopressata** | 威尼托/卡拉布里亚/托斯卡纳 | 用猪的碎肉（非精选部位）压制成型的salami，口感更粗犷 |

## 十一、奶酪（Formaggi）

意大利的制酪传统可追溯到罗马时代之前。19世纪，各个地区的特色奶酪已经高度成熟。

### 23. 主要奶酪种类

| 奶酪 | 地区 | 描述 |
|------|:--:|------|
| **Parmigiano-Reggiano（帕尔马-雷焦干酪）** | 艾米利亚-罗马涅 | **意大利奶酪之王**，历史可追溯至12世纪。生牛奶制，大轮（约40kg），至少熟成12–24个月甚至更久。19世纪已经形成严格的地理保护产区。用途：磨碎撒在意面/汤/risotto上，切小粒作为"grana"零食，或刨薄片配梨/aceto balsamico |
| **Grana Padano** | 伦巴第/威尼托/皮埃蒙特/特伦蒂诺 | 帕尔马干酪的"表亲"，产区更广，法规略宽松，风味更温和 |
| **Mozzarella di Bufala Campana（水牛莫扎瑞拉）** | 坎帕尼亚 | 水牛奶制成的新鲜拉伸凝乳（pasta filata）奶酪。质地柔软有弹性，内藏乳清，咬开会流出白色牛乳汁。19世纪主要在产区消费，极不易运输保存 |
| **Fior di Latte（牛乳莫扎瑞拉）** | 各地 | 用牛奶替代水牛奶制作的莫扎瑞拉，成本更低，19世纪中后期开始大规模生产 |
| **Ricotta（里科塔）** | 各地 | 从制作奶酪剩下的乳清（siero di latte）中二次提取出来的絮状凝乳。严格来说不算奶酪，而是"乳清奶酪"。新鲜版湿润柔软，盐渍版（ricotta salata）可磨碎用于意面。大量用于填馅（ravioli, cannoli, cassata） |
| **Pecorino Romano（罗马羊奶酪）** | 撒丁岛（古代罗马的产区在拉齐奥，但在19世纪撒丁岛已成主产区） | 全羊奶硬质奶酪，味咸而强劲。是罗马cacio e pepe、carbonara等经典酱料的标配 |
| **Pecorino Toscano（托斯卡纳羊奶酪）** | 托斯卡纳 | 比romano更温和，熟成期更短，有时用迷迭香等香料裹皮。既可以磨碎入菜也可以切片配梨和蜂蜜直接吃 |
| **Gorgonzola（戈尔贡佐拉蓝纹奶酪）** | 伦巴第 | 牛乳蓝纹软奶酪，青霉（penicillium glaucum）形成蓝绿斑纹。分dolce（甜味/乳脂感）和piccante（辛辣味/熟成更久）两种。历史将近千年，名字来自米兰附近的小镇戈尔贡佐拉 |
| **Fontina（芳提娜）** | 瓦莱达奥斯塔 | 生牛乳半硬质奶酪，风味浓郁，加热后融化性极好。是fonduta（意式奶酪火锅）和polenta concia的灵魂 |
| **Taleggio** | 伦巴第 | 牛乳软质洗皮奶酪，橙粉色外皮，内里奶味浓郁，略带酸味 |
| **Mascarpone（马斯卡彭）** | 伦巴第 | 奶油+柠檬酸加热凝固的超高脂鲜奶酪，用于提拉米苏（但提拉米苏本身是20世纪60年代的发明，19世纪不存在）、混合水果或做risotto的accents |
| **Burrata（布拉塔）** | 普利亚 | Mozzarella皮囊内填奶油+stracciatella（撕碎的mozzarella条）。据说发明于20世纪初（约1920年代），19世纪末可能已有早期版本 |
| **Caciocavallo** | 南部（巴西利卡塔/普利亚/西西里） | 悬挂熟成的拉伸凝乳奶酪，呈葫芦形或梨形，用绳子系挂 |
| **Provolone** | 南部（坎帕尼亚/巴西利卡塔） | 大型拉伸凝乳硬质奶酪，烟熏或非烟熏版本，形状从小到大不一 |

## 十二、面包（Pane）

意大利的面包文化极其多元，每个城市都有自己的传统面包。19世纪尤为如此——彼时还没有大型工业面包厂，每个街区的面包房都有自己独特的发酵剂和配方。

### 24. 主要面包种类

| 面包 | 地区 | 描述 |
|------|:--:|------|
| **Pane Toscano（托斯卡纳面包）** | 托斯卡纳 | **无盐面包**。传说源于中世纪比萨和佛罗伦萨的盐税——佛罗伦萨人拒绝加盐以示反抗，从此形成无盐传统。外脆内松孔大量、微酸的天然酵母面包。无盐的面包被称为"sciocco"（傻的）。为什么无盐？因为托斯卡纳菜（如prosciutto crudo、salame、鸡肝酱crostini）本身已经够咸 |
| **Pane di Altamura** | 普利亚（阿尔塔穆拉） | 用当地硬质小麦（grano duro）粗磨面（semola rimacinata）手工制作的大圆面包，天然酵母（madre acida）发酵，木柴烤炉烘烤。外壳极厚极硬，内部金黄柔软，可保存1–2周不变质——牧人和农民背着下地干活 |
| **Focaccia Genovese（热那亚佛卡夏）** | 利古里亚 | 用大量特级初榨橄榄油制成的厚软扁平面包，表面用手指戳出小洞，填满粗海盐和水+橄榄油的乳液（salamoia）。香脆的外壳和湿润的内心之间是极致对比。通常不加topping——橄榄油本身已是足够的奢侈 |
| **Focaccia alla Messinese / Siciliana** | 西西里 | 西西里版的佛卡夏，更厚，可能加入番茄、鳀鱼、洋葱，类似厚版的pizza bianca |
| **Grissini Torinesi（都灵面包棒）** | 皮埃蒙特 | 手指粗细的脆面包棒（有些版本长达臂展），极干极脆。传说由都灵医生为消化不良的萨伏依公爵发明，名字来自皮埃蒙特方言"ghërsa"（长棍）。19世纪都灵的grissini已经作为高级餐点出口全欧 |
| **Ciabatta** | 威尼托 | **重要说明：ciabatta（拖鞋面包）不是19世纪的产物！** 它于1982年由威尼托的面包师发明，作为对法式baguette的意大利回应。在19世纪的文献和Artusi的书中均无记载 |
| **Pane Casereccio / Pane di Casa（家常面包）** | 各地 | 每个村庄都有自己的版本。通常为天然酵母发酵、大圆形或椭圆形，由本地面包房每天早晨用柴烤炉批量生产 |
| **Pane di Semola / Pane Siciliano** | 西西里 | 硬质小麦粗面制作的硬皮面包，撒芝麻。西西里气候干燥炎热，硬质的semola面包适合长期保存 |
| **Michetta / Rosetta** | 伦巴第（米兰） | 米兰版的小圆面包——内部几乎中空，只有薄薄一层脆壳。奥匈帝国统治时期传入的技艺（奥地利Kaisersemmel的意大利化），用手撕开，中间填入salami或奶酪即可带着上班 |
| **Friselle** | 普利亚 | 环状的硬面饼干（先烘烤后切开再烤），极度干燥，吃前需在流水下冲软，淋番茄、橄榄油、盐和罗勒。渔夫和农民便于携带的口粮 |
| **Piadina Romagnola** | 罗马涅 | 扁平的未发酵面饼（面粉+猪油/橄榄油+盐+水），在陶盘（testo）或铁板上煎烤。当场做好趁热折叠，内夹squacquerone奶酪（罗马涅的软鲜奶酪）、prosciutto crudo和芝麻菜（rucola）。19世纪已经是罗马涅人的日常面包替代品，路边小摊（piadineria）随处可见 |
| **Crescia / Torta al Testo** | 翁布里亚/马尔凯 | 类似piadina但更厚的铁板烤饼，可切开成口袋填入火腿和蔬菜 |
| **Pane Carasau / Carta da Musica（音乐纸面包）** | 撒丁岛 | 极薄的脆烤麦饼（直径约30–40cm，几乎透明），因薄如羊皮纸/乐谱纸而得名。可以保存数月，撒丁岛牧羊人带着上山放牧数月无需补给 |

## 十三、甜点与糕点（Dolci e Pasticceria）

### 25. 经典节日甜点

| 甜点 | 地区 | 描述 |
|------|:--:|------|
| **Panettone** | 米兰/伦巴第 | 米兰圣诞节标志：高圆柱形甜面包，含大量黄油+蛋黄+糖渍橙皮（canditi）+葡萄干。天然酵母发酵，制作耗时数十小时。关于其起源的传说很多——最常见的是：15世纪米兰Ludovico il Moro公爵府上的甜品师弄砸了圣诞宴会的甜点，年轻的厨房学徒Toni将自己的私藏——加了黄油、鸡蛋、蜜饯的面团——救场烤成面包，公爵问这是什么，厨师说"il pan del Toni"（Toni的面包），便成了"panettone"。19世纪已经在米兰面包房商业化生产 |
| **Pandoro** | 维罗纳/威尼托 | 金字塔形（八角星截面）的黄油面包，无任何果仁或蜜饯，纯以蛋、黄油、香草为原料。出炉后洒大量香草糖粉（糖粉+香草籽）模拟雪山。名字意为"金色面包"，是维罗纳对米兰panettone的回应。19世纪末已有规模生产 |
| **Torrone** | 克雷莫纳（Cremona）/伦巴第 | 蜂蜜+蛋白+杏仁/开心果+香草制成的硬质牛轧糖。克雷莫纳在19世纪已经以其torrone闻名全国，圣诞期间全意大利都有售 |
| **Mostarda di Cremona** | 克雷莫纳/伦巴第 | **不是芥末！** 这是用糖浆和芥末精油（essenza di senape）浸渍的整颗水果（樱桃、梨、杏、无花果、橙皮），强烈的辛辣与甜味碰撞。配bollito misto（煮肉）食用。是皮埃蒙特和伦巴第冬季餐桌的灵魂调味品 |
| **Struffoli** | 那不勒斯/坎帕尼亚 | 油炸小蜂蜜面团球，堆成金字塔形或花环形，淋大量蜂蜜，撒彩色糖珠（diavolilli / codette）。那不勒斯圣诞节的标志甜点 |

### 26. 复活节与狂欢节甜点

| 甜点 | 地区 | 描述 |
|------|:--:|------|
| **Colomba Pasquale（复活节鸽形蛋糕）** | 伦巴第 | 与panettone同类的酵母甜面包，做成鸽子（和平与复活）的形状，表面撒杏仁糖霜（glassa di mandorle）和杏仁片。19世纪末新创，很快成为全国复活节的标志 |
| **Pastiera Napoletana** | 那不勒斯 | 煮熟的小麦粒（grano cotto）+ ricotta奶酪 + 鸡蛋 + 蜜饯橙皮 + 橙花水（acqua di fiori d'arancio）制成的浓馅挞，外面是甜酥皮（pasta frolla）。在周四圣日（Giovedì Santo）前做好准备，让风味在复活节周日完美融合。这是那不勒斯复活节的灵魂甜点，可追溯到古希腊时期（以小麦象征生命的重生） |
| **Casatiello / Tortano** | 那不勒斯 | 咸味复活节面包/蛋糕——将salami、猪油、pecorino、黑胡椒嵌入发酵面团中烤制，表面十字排列整颗鸡蛋（带壳烤熟）。是复活节周一郊游（Pasquetta）的经典携带食品 |
| **Chiacchiere / Frappe / Bugie / Cenci / Galani** | 各地（全国狂欢节食品，每地名称不同） | 炸薄面片（约1mm薄），撒大量糖粉。狂欢节（Carnevale，大斋前）的标志甜点——狂欢节后就是40天的斋期，所以要尽情消耗糖、蛋、油脂 |
| **Castagnole** | 各地 | 油炸小甜面团球，内软外微脆，可填入custard（crema pasticcera），撒糖粉。狂欢节食品 |
| **Frittelle di Riso（威尼斯米饼）** | 威尼斯 | 米+牛奶+糖+鸡蛋煮成稠糊，加松子+葡萄干+蜜饯水果+朗姆酒，做成小圆饼油炸 |
| **Zeppole di San Giuseppe（圣约瑟夫炸/烤泡芙）** | 那不勒斯/坎帕尼亚 | 3月19日（圣约瑟夫节 = 意大利的父亲节）的泡芙圈。油炸或烤制，中间填入蛋奶酱（crema pasticcera），表面放一颗酸樱桃（amarena）。19世纪那不勒斯的面包房在这个日子门口排长队 |

### 27. 西西里甜点

| 甜点 | 地区 | 描述 |
|------|:--:|------|
| **Cannoli Siciliani（西西里奶酪卷）** | 西西里（巴勒莫为首） | 油炸酥脆管状外壳（scorza），内填新鲜ricotta di pecora（绵羊奶ricotta）+ 糖 + 巧克力碎 + 蜜饯南瓜/橙皮。在嘉年华期间食用。Cannoli的成型需要专用的金属管（cannello）——炸好的面壳裹在管上形成中空，吃前才灌入ricotta（防止壳软化）。19世纪巴勒莫的修道院修女已将cannoli的配方做到了极致 |
| **Cassata Siciliana（西西里卡萨塔蛋糕）** | 西西里（巴勒莫） | 极尽奢华的蛋糕：绿色杏仁面皮（pasta di mandorle）、ricotta奶酪 + 糖 + 巧克力碎的内馅、海绵蛋糕（pan di Spagna）基底、表面覆盖糖霜（glassa di zucchero）和蜜饯水果的精美排布。巴勒莫修女院的巅峰造诣。名字来自阿拉伯语"qas'ah"（碗/圆形模具），见证了西西里阿拉伯时代（9–11世纪）的烹饪遗产 |
| **Granita Siciliana（西西里冰沙）** | 西西里/南部 | 水+糖+水果（柠檬、咖啡、杏仁、草莓、桑葚等）的半冻状态冰晶，质地比sorbet更粗糙，用小匙舀食。西西里人早餐吃咖啡granita配热的brioche col tuppo（圆顶甜面包）——一面冰一面热，夏日早餐的终极享受。19世纪街头巷尾的granita摊子遍布西西里大小城镇 |
| **Frutta di Martorana（马尔托拉纳修女院杏仁水果）** | 西西里（巴勒莫） | 杏仁粉+糖+水做成杏仁膏（marzapane），手工捏塑成各种水果形状（桃子、无花果、橘子、柠檬、石榴等），以植物色素染色。成品栩栩如生，几可乱真。巴勒莫马尔托拉纳修道院的修女在19世纪已经将这门技艺传承了数百年 |

### 28. 其他经典甜点

| 甜点 | 地区 | 描述 |
|------|:--:|------|
| **Zabaione / Zabaglione** | 皮埃蒙特/伦巴第 | 蛋黄+糖+甜酒（通常为Marsala）隔水加热搅打至蓬松如慕斯状的暖甜羹。可以直接热食（用匙舀），也可倒在水果上，或做成冷冻版。Artusi记录了经典配方 |
| **Panna Cotta** | 皮埃蒙特 | 奶油+糖+香草煮开后加明胶（gelatina）凝固成布丁的甜品。最简单的原料，最醇厚的口感。配上焦糖、巧克力、或新鲜浆果（frutti di bosco）酱汁。皮埃蒙特的cucina povera甜品典范 |
| **Crostata di Frutta / Marmellata（水果挞/果酱挞）** | 各地 | 甜酥皮（pasta frolla）+ 果酱（marmellata，杏/樱桃/黑莓）或新鲜水果 + 蛋奶酱（crema pasticcera）的经典开放式挞。国民级的家庭烘焙甜点 |
| **Torta della Nonna（祖母蛋糕）** | 托斯卡纳 | 两层甜酥皮（pasta frolla）夹柠檬味蛋奶酱（crema pasticcera）的圆形挞，表面撒松子和糖粉 |

| **Tiramisù（提拉米苏）** | **重要：19世纪不存在！** | 提拉米苏（咖啡浸泡的手指饼干+ mascarpone奶酪+可可粉层叠甜点）是1960–70年代在威尼托发明的，尽管其确切的发明地（特雷维索vs弗留利）有争议。Artusi的书中没有任何关于tiramisu的记录。但mascarpone奶酪、手指饼干（savoiardi）等组成元素在19世纪已个别存在 |
| **Savoiardi（萨伏依手指饼干）** | 皮埃蒙特（萨伏依） | 极其松脆的蛋糖海绵饼干，形状如手指，单独食用或浸咖啡/酒/做蛋糕夹层 |

## 十四、冰淇淋与雪葩（Gelato e Sorbetto）

意大利是冷冻甜点的故乡——古代罗马人用山顶的积雪拌蜂蜜和水果做成的冷食，经由阿拉伯人传入西西里的雪葩（sharbat→sorbetto）技术，最终在文艺复兴的意大利宫廷中演变为我们今天所知的gelato和sorbetto。

### 29. 19世纪的Gelato

到了19世纪，gelato已经不是贵族专属——街头gelato摊（gelataio）在大城市中已经相当普遍。Artusi的书中有大量gelato和sorbetto的配方。

| 类型 | 描述 |
|------|------|
| **Gelato di Crema（奶油冰淇淋）** | 基础：全脂牛奶+鲜奶油+蛋黄+糖+香草。蛋黄使质地更加丝滑浓郁，是意大利gelato区分于法式冰淇淋的灵魂 |
| **Gelato di Cioccolato（巧克力冰淇淋）** | 热巧克力（cacao in polvere）混入基础crema中 |
| **Gelato di Caffè（咖啡冰淇淋）** | 浓缩咖啡（caffè ristretto）混入基础crema |
| **Gelato di Pistacchio（开心果冰淇淋）** | 西西里勃朗特（Bronte）产的开心果磨酱混入奶浆。这座埃特纳火山山坡上的小镇在19世纪已是世界级的开心果产地 |
| **Gelato di Nocciola（榛子冰淇淋）** | 皮埃蒙特（朗格Langhe地区）的榛子（tonda gentile品种）烘焙后研磨入奶浆 |
| **Sorbetto di Limone（柠檬雪葩）** | 不含奶——柠檬汁+糖浆+水+蛋清（有时）搅打冷冻。口感清爽，用作两道菜之间的清口（palate cleanser）|
| **Sorbetto di Caffè / Fragola / Lampone** | 咖啡/草莓/覆盆子各种变体 |
| **Semifreddo** | 介于mousse和gelato之间的半冷冻甜点——不需搅拌机，在模具中冷冻成型，口感更轻软 |

**Artusi的Gelato智慧：** 1891年的书中解释说，在没有现代制冷设备的年代，gelato的制作依赖手动旋转的金属桶放置在冰+盐的混合物中（盐降低冰点），一边转桶一边用刮刀（spatola）刮下桶壁的冰结晶，防止形成大冰渣。一桶gelato可能需要一整个小时的手动搅拌。

## 十五、饮品（Bevande）

### 30. 咖啡（Caffè）

**意大利语中的"caffè"默认指espresso**——这一习惯在19世纪已经开始形成。

- **Espresso的诞生**（1884年）：都灵的Angelo Moriondo获得了第一台蒸汽驱动咖啡机的专利——这台机能快速制造浓缩咖啡（espresso = 快速压出的）。不过它的形态与今天的espresso还不完全相同。
- **现代espresso机的雏形**（1901年）：米兰的Luigi Bezzera进一步改进了机器（增加了压力泵），1903年Desiderio Pavoni买下专利并商业化生产——这就是La Pavoni品牌的开端。虽然严格来说时间是1901年（仅跨出19世纪一步），但19世纪末的咖啡馆已经充满了对"快速浓缩咖啡"的狂热技术追求。
- **19世纪的咖啡文化**：在那不勒斯、罗马、威尼斯、都灵、米兰、佛罗伦萨、的里雅斯特等城市，咖啡馆（caffè）是知识分子、艺术家、革命者的聚集地。不同城市的咖啡习惯有所不同——那不勒斯人偏爱用"cuccumella"（翻转式那不勒斯咖啡壶）做家常咖啡，威尼斯则有全景式广场咖啡馆（Caffè Florian, 1720年开业；Caffè Quadri）。

### 31. 葡萄酒（Vino）

意大利葡萄酒在19世纪经历了革命性的变化：

| 酒 | 地区 | 描述 |
|------|:--:|------|
| **Barolo（巴罗洛）** | 皮埃蒙特（朗格Langhe） | 19世纪中叶之前，Barolo是一种甜红葡萄酒（由于在冷天发酵不完全留下残糖）。法国酿酒师Louis Oudart（受卡米罗·奔馳伯爵之邀）在1840年代将干红发酵技术带入Barolo，现代版的"王者之酒、酒之王者"由此诞生 |
| **Chianti** | 托斯卡纳 | 19世纪Chianti的配方（由Bettino Ricasoli男爵于1872年标准化）是：Sangiovese（70%+）+ Canaiolo（15%）+ Malvasia bianca（10%）+ 也可能有Colorino等品种。装入fiasco（用稻草包裹的圆底玻璃大肚瓶）是19世纪Chianti的标志性包装 |
| **Brunello di Montalcino** | 托斯卡纳 | Ferruccio Biondi-Santi于1888年首次以单一的Sangiovese Grosso品种（当地称Brunello）酿出可长期陈年的醇厚红酒 |
| **Marsala** | 西西里 | 19世纪是Marsala的黄金时代——英国商人John Woodhouse和后来的Ingham-Whitaker家族发现了这种加强型葡萄酒（类似雪利/波特），将其推向英国和全球市场。Marsala在Artusi的厨房中是zabaione和甜点的核心配料 |
| **Vermouth** | 皮埃蒙特（都灵） | 以白葡萄酒为基，加入苦艾（assenzio / artemisia）等数十种药草和香料浸泡调味的加香型加强酒。1786年Antonio Benedetto Carpano在都灵创造了第一款商业vermouth。19世纪都灵成为世界vermouth之都（Martini & Rossi 1863年、Cinzano 1757年等品牌均起于此） |
| **Prosecco** | 威尼托 | 19世纪已有酿造，但现代Prosecco的繁荣（用Glera葡萄的Charmat法）是20世纪的事 |
| **Lambrusco** | 艾米利亚-罗马涅 | 微泡红葡萄酒，19世纪时多为甜型或半甜型（amabile），配艾米利亚丰富的猪肉菜肴 |
| **Sfuso / Vino da Tavola（散装餐酒）** | 各地 | 绝大多数意大利人在19世纪喝的是直接从酒桶中打的散装葡萄酒（vino sfuso），便宜、低酒精度、在本地消费。农民每餐不可或缺——"面包和水使人活下去，酒使人活得有滋味" |

### 32. 酒精饮品

| 饮品 | 描述 |
|------|------|
| **Grappa** | 用榨过汁的葡萄皮渣（vinaccia）蒸馏出来的高度烈酒（40–60%）。19世纪是意大利最普遍的烈酒，家家户户可能都有自己的蒸馏器具。威尼托的Bassano del Grappa是核心产区 |
| **Amaro（苦味药草利口酒）** | 各种草药、树根、树皮、柑橘皮浸泡在烈酒或葡萄酒中，加糖制成的餐后助消化酒（digestivo）。各地皆有本地的amaro配方——往往来自修道院或药房 |
| **Limoncello** | 柠檬皮浸泡在纯酒精中萃取精油，再兑入糖浆水。坎帕尼亚（阿马尔菲海岸和索伦托的sfusato柠檬）为中心产区。19世纪作为家常自制的餐后酒存在 |
| **Sambuca** | 茴香风味的透明甜利口酒，拉齐奥起源 |
| **Strega（女巫酒）** | 贝内文托（坎帕尼亚）的黄色药草利口酒，含藏红花、薄荷、杜松、肉桂等约70种原料 |

### 33. 非酒精饮品

| 饮品 | 描述 |
|------|------|
| **Acqua（水）** | 19世纪的意大利城市中，清洁饮水并非理所当然。罗马有引水渠供给的公共喷泉（fontanella / nasone"大鼻子"），水质优良。农村地区依赖井水和泉水 |
| **Gazzosa / Gassosa** | 柠檬味甜碳酸饮料，是19世纪儿童和戒酒者的饮料 |
| **Orzata（杏仁糖浆水）** | 大麦（orzo）或苦杏仁制成的甜糖浆兑水，夏日消暑饮料 |
| **Latte di Mandorla（杏仁奶）** | 西西里和普利亚的杏仁磨浆+糖+水的饮料，斋戒期也可饮用 |

## 十六、主要地区菜系特色速览

意大利的统一（Risorgimento）在1861年才实现，因此"意大利菜"在19世纪的大多数时间里是**一套地方传统的松散家族**，而非统一菜系。Artusi的伟大贡献之一就是将这些地方传统汇集在一本书中，为"意大利民族菜"的形成奠定了基础。

| 地区 | 核心特征 | 标志性食材 | 代表菜品 |
|------|------|------|------|
| **皮埃蒙特（Piemonte）** | 法餐影响最深的地区（萨伏依王室与法国血脉）、丰富黄油和奶酪、白松露（阿尔巴）、牛肉 | Barolo红酒、白松露、榛子、fontina奶酪、小牛肉 | Vitello Tonnato, Bollito Misto, Brasato al Barolo, Agnolotti, Risotto Piemontese, Panna Cotta, Grissini |
| **伦巴第（Lombardia）** | 黄油和奶油为主（不用橄榄油）、米+Risotto文化、玉米粥Polenta核心 | 黄油、arborio/carnaroli米、藏红花、帕尔马干酪、牛骨髓、小牛肉 | Risotto alla Milanese, Ossobuco, Cotoletta alla Milanese, Panettone, Cassoeula, Gorgonzola |
| **威尼托（Veneto）** | 潟湖海鲜、鳕鱼干（stoccafisso/baccalà）贸易、Risi e Bisi米豌豆文化 | 鳕鱼干、墨鱼、riso（大米）、radicchio（红菊苣）、polenta | Baccalà alla Vicentina, Risi e Bisi, Fegato alla Veneziana, Sarde in Saor, Pandoro |
| **利古里亚（Liguria）** | 热那亚海洋共和国调味传统、香草与橄榄油的核心地位、"绿色"菜肴 | 罗勒、松子、橄榄油、鳀鱼、鹰嘴豆粉、核桃 | Pesto alla Genovese, Trenette al Pesto, Focaccia Genovese, Farinata, Salsa di Noci |
| **艾米利亚-罗马涅（Emilia-Romagna）** | **意大利最肥沃的"美食谷"**（Food Valley）、手工鸡蛋面、猪肉加工荟萃 | 帕尔马干酪（Parmigiano-Reggiano）、prosciutto di Parma、mortadella、aceto balsamico（巴萨米克醋）、鸡蛋 | Tagliatelle al Ragù Bolognese, Tortellini in Brodo, Lasagne, Piadina, Zampone/Cotechino |
| **托斯卡纳（Toscana）** | Cucina Povera的圣地——极简、不浪费、尊重食材原味。无盐面包 | pane sciocco（无盐面包）、橄榄油、白腰豆（cannellini）、bistecca牛肉、野味 | Bistecca alla Fiorentina, Ribollita, Pappa al Pomodoro, Cantucci + Vin Santo, Lampredotto, Panforte |
| **拉齐奥（Lazio）——罗马** | 内脏（quinto quarto）= 屠宰场废弃物文化、pecorino romano羊奶酪、guanciale | pecorino romano、guanciale、朝鲜蓟、牛肚、puntarelle | Pasta alla Carbonara（19世纪末雏形）, Cacio e Pepe, Amatriciana, Abbacchio, Carciofi alla Giudia, Trippa, Saltimbocca |
| **坎帕尼亚（Campania）——那不勒斯** | 番茄面食文化的堡垒、披萨故乡、莫扎瑞拉水牛奶酪、长面食 | San Marzano番茄、水牛mozzarella、Semolina硬质面、柠檬、海鲜 | Pizza Margherita, Spaghetti al Pomodoro, Ragù Napoletano, Pastiera, Frittatine, Mozzarella in Carrozza |
| **普利亚（Puglia）** | 小麦和橄榄油大区、耳朵面orecchiette、蔬菜为主 | orecchiette、cime di rapa、burrata、橄榄油、硬质小麦 | Orecchiette con Cime di Rapa, Pane di Altamura, Fave e Cicoria（蚕豆泥+苦菊） |
| **西西里（Sicilia）** | 阿拉伯+诺曼+西班牙+希腊文化层积的复合菜系，甜酸（agrodolce）广泛使用 | 茄子、剑鱼、沙丁鱼、柑橘、杏仁、开心果、ricotta、松子、葡萄干 | Caponata, Pasta alla Norma（茄子+ricotta salata）, Cannoli, Cassata, Arancini（炸米球）, Pane con la Milza（牛脾三明治） |
| **撒丁岛（Sardegna）** | 内陆牧羊文化、极少的面食（以面包替代）、羊奶奶酪 | pecorino sardo、pane carasau、烤乳猪、羊肉、野香草 | Porceddu（烤乳猪）、Malloreddus（撒丁小面疙瘩）、Pane Carasau、Seadas（蜂蜜奶酪炸饼） |
| **弗留利-威尼斯朱利亚（Friuli-Venezia Giulia）** | 中欧交汇——德国/奥地利/斯拉夫影响与威尼斯海鲜传统的融合 | 玉米粥、酸菜、辣根、ham | Jota（酸菜豆子汤）、Goulash Triestino（的里雅斯特匈牙利炖牛肉） |
| **特伦蒂诺-上阿迪杰（Trentino-Alto Adige）** | 奥地利-德意志烹饪传统混合意大利面食，山区饮食 | speck、苹果、玉米粥、canederli（面包丸）| Canederli in Brodo（肉汤面包丸）, Speck Alto Adige, Strudel（苹果卷） |

## 十七、19世纪意大利经典食谱书籍

| 书名 | 作者 | 初版 | 意义 |
|------|------|:--:|------|
| **《厨房科学与饮食艺术》（La scienza in cucina e l'arte di mangiar bene）** | Pellegrino Artusi（1820–1911） | 1891 | **没有此书就没有现代意大利菜。** 初版含475道菜谱，由作者自费出版（因无出版社看好）。到1910年（Artusi去世前一年）共出版15版，增补至790道菜谱。该书将意大利各地的烹饪传统首次汇集在同一个框架下，以标准化的意大利语书写（而非当地方言），为统一后寻找民族身份认同的意大利人提供了"味觉上的统一"。Artusi本人是佛罗伦萨的丝绸商人（非厨师），自嘲为"烹饪爱好者"，但他用一生旅行收集的菜谱加上科学的实验精神，造就了意大利料理的圣经 |
| **《那不勒斯烹饪》（Cucina Napoletana）** | Ippolito Cavalcanti（Duca di Buonvicino） | 1837（首版）、1839（增订版*Cucina teorico-pratica*） | 19世纪上半叶最重要的南方菜谱书。详尽记录了那不勒斯菜系的300余道菜谱，包含pizza、番茄酱意面、海鲜料理等。他首次在书中区分了"cucina aristocratica"（贵族菜，含法餐影响）和"cucina casareccia"（家常菜，平民菜） |
| **《伦巴第-威尼托烹饪》（Il Cuoco Lombardo-Veneto）** | Giovanni Vialardi | 1854 | 萨伏依王室厨师长。记录了北意大利贵族宴会的菜谱，但也包含大量中产家庭的日常菜 |
| **《皮埃蒙特烹饪》（La Cuciniera Piemontese）** | 多名作者 | 1798（初版），多次再版 | 记录了18世纪末至19世纪皮埃蒙特的烹饪传统 |
| **《烹饪之王》（Il Re dei Cuochi）** | Giovanni Nelli | 1868 | 罗马流行的大型菜谱汇编，反映中部意大利的饮食 |
| **《现代烹饪家》（Il Cuoco Moderno）** | 多人汇编 | 19世纪末 | 大量印行的实用烹饪手册之一 |

## 附录：19世纪意大利各阶级日常饮食速览

| 社会阶级 | 日常饮食概况 |
|:--|------|
| **贵族与富裕资产阶级** | 受法餐影响——法文菜单、全套法式上菜顺序。正餐：开胃菜（salumi/奶酪）+ 两道primo（一道汤+一道面/risotto）+ 一道鱼 + 一道肉 + contorno（蔬菜配菜）+ dolce（甜点）+ 水果 + 咖啡 + 利口酒。野味（selvaggina）是地位的象征。酒以进口法国酒为主，也有本地名酒（Barolo, Chianti）。饮食中的肉量远远超过社会平均值。雇佣私人厨师或法国主厨 |
| **城市中产（商人、专业人士、公务员）** | 1861年统一后日渐增长的阶级。Artusi正是为他们而写。一天的饮食（以托斯卡纳为例）：早餐（caffelatte + 面包）、午餐（primo如面汤/risotto + secondo如肉/鱼 + contorno + 水果）、晚餐（更轻，可能只是汤/面 + 冷肉 + 奶酪）。周末和节日的午餐更加丰盛。肉类约每周3–4次。咖啡是日常 |
| **城市工人/工匠** | 早餐：面包 + caffè（或caffè d'orzo大麦咖啡——咖啡太贵买不起咖啡豆时）。午餐：minestra（汤/面）是全天的主餐，可能配面包和一片奶酪或几片salami。"Pranzo"（午餐）一词对于工人来说就是指那一碗minestra。晚餐：剩汤+面包，或polenta+蔬菜（北部）。肉是稀罕物——每周一次或更少，且是内脏或香肠末端的廉价部位。那不勒斯工人可能在大街上买一片披萨 |
| **农村农民（Contadini）** | 占19世纪意大利总人口绝大多数（1861年约70%农业人口）。饮食被cucina povera定义。**北部农民**：polenta（玉米粥）+ 少许奶酪或鳀鱼，可能一天两顿polenta（这也是糙皮病/烟酸缺乏症的根源）。汤以豆子、卷心菜、土豆、面包块为基底。**中部农民**：无盐面包+橄榄油+洋葱+豆子汤。肉仅在节日食用（通常猪在冬季宰杀，肉制品（salami/ham）要支撑一整年）。**南部农民**：面包+番茄+橄榄油+大蒜+pasta（硬质干面）+ 豆类。面包可能一周烤一次，到周末已经硬如石头，需要浸汤才能食用。鱼对沿海农民来说相对容易获取 |
| **最底层（日薪劳动者、无地雇农）** | 一天可能只有一顿真正的热食——一碗汤（acqua+ stale bread + 一点蔬菜）。大量的食用野生可食植物（野菜cicoria selvatica、荨麻ortic、蒲公英、野蒜）。18–19世纪意大利多次爆发饥荒（如1816–17年），那时的"面包"可能混入栗子粉甚至树皮粉。儿童死亡率极高——部分原因就是营养不良 |
| **修道院/修女院饮食** | 全年大量斋戒日。修道院的素食传统意外地创造了意大利菜中许多经典：杏仁膏水果（frutta di martorana）、许多蛋糕配方和利口酒配方均来自修女院。修道院酿造的啤酒和药草利口酒在19世纪的小城镇非常重要 |
